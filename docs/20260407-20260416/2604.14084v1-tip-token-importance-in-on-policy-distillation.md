## 摘要
同策略知识蒸馏（OPD）在教师模型的 Token 级监督下，利用学生模型自身的采样序列（rollouts）进行训练。并非所有 Token 位置都具有同等的重要性，但现有的 Token 重要性观点尚不完整。我们提出了一个直接的问题：在 OPD 中，哪些 Token 携带了最有用的学习信号？我们的回答是，信息丰富的 Token 主要来自两个区域：学生模型熵值较高的位置，以及学生模型熵值较低但师生散度（divergence）较高的位置（即学生模型过度自信且出错的位置）。实验表明，学生模型的熵是一个强有力的一阶代理指标：基于熵的采样仅保留 50% 的 Token 即可达到或超过全量 Token 训练的效果，同时减少高达 47% 的峰值显存。然而，仅靠熵会遗漏第二个重要区域。当我们隔离出低熵、高散度的 Token 时，仅在不足 10% 的 Token 上进行训练就几乎能达到全量 Token 基准的效果，这表明过度自信的 Token 携带了密集的纠错信号，尽管它们在仅基于熵的规则下几乎不可见。我们通过 TIP（同策略蒸馏中的 Token 重要性）整理了这些发现，这是一种基于学生模型熵和师生散度的双轴分类法，并从理论上解释了为什么熵虽然有用但在结构上是不完整的。这一观点启发了结合不确定性（uncertainty）和分歧（disagreement）的类型感知 Token 选择规则。我们在 MATH-500、AIME 2024/2025 以及用于长程智能体规划的 DeepPlanning 基准测试上，针对涵盖 Qwen3、Llama 和 Qwen2.5 的三组师生模型对验证了这一结论。在 DeepPlanning 中，仅在不足 20% 的 Token 上进行训练就超过了全量 Token 的 OPD。我们的实验通过扩展 OPD 仓库实现，该仓库支持在有限的 GPU 预算下对大型模型进行显存高效的蒸馏。

## Abstract
On-policy knowledge distillation (OPD) trains a student on its own rollouts under token-level supervision from a teacher. Not all token positions matter equally, but existing views of token importance are incomplete. We ask a direct question: which tokens carry the most useful learning signal in OPD? Our answer is that informative tokens come from two regions: positions with high student entropy, and positions with low student entropy plus high teacher--student divergence, where the student is overconfident and wrong.   Empirically, student entropy is a strong first-order proxy: retaining $50\%$ of tokens with entropy-based sampling matches or exceeds all-token training while reducing peak memory by up to $47\%$. But entropy alone misses a second important region. When we isolate low-entropy, high-divergence tokens, training on fewer than $10\%$ of all tokens nearly matches full-token baselines, showing that overconfident tokens carry dense corrective signal despite being nearly invisible to entropy-only rules.   We organize these findings with TIP (Token Importance in on-Policy distillation), a two-axis taxonomy over student entropy and teacher--student divergence, and give a theoretical explanation for why entropy is useful yet structurally incomplete. This view motivates type-aware token selection rules that combine uncertainty and disagreement. We validate this picture across three teacher--student pairs spanning Qwen3, Llama, and Qwen2.5 on MATH-500 and AIME 2024/2025, and on the DeepPlanning benchmark for long-horizon agentic planning, where Q3-only training on $<$$20\%$ of tokens surpasses full-token OPD. Our experiments are implemented by extending the OPD repository https://github.com/HJSang/OPSD_OnPolicyDistillation, which supports memory-efficient distillation of larger models under limited GPU budgets.

#### 1. 核心问题与整体含义（研究动机和背景）
在同策略知识蒸馏（On-Policy Distillation, OPD）中，学生模型根据自身生成的序列（rollouts）接受教师模型的逐 Token 指导。然而，并非所有 Token 对学习的贡献都相同。现有的方法往往对所有 Token 均匀施加损失，这不仅浪费计算资源，还可能引入冗余信号。
**核心问题**：在 OPD 过程中，哪些 Token 携带了最有价值的学习信号？
**研究动机**：通过识别关键 Token，可以在保证甚至提升模型性能的前提下，大幅降低训练时的显存占用和计算开销，解决大模型蒸馏中 GPU 资源受限的痛点。

#### 2. 方法论：TIP 分类法与 Soft-OR 评分
论文提出了 **TIP (Token Importance in on-Policy distillation)** 框架，将 Token 重要性归纳为基于“学生熵（$h_t$）”和“师生分歧（$\delta_t$）”的双轴分类法：
*   **核心思想**：将 Token 空间划分为四个象限：
    *   **Q1 (高熵+高分歧)**：学生不确定且出错，信号最密集。
    *   **Q2 (高熵+低分歧)**：学生不确定但基本正确，提供稳定信号。
    *   **Q3 (低熵+高分歧)**：**盲点区域**。学生过度自信但错误，携带关键的纠错信号。
    *   **Q4 (低熵+低分歧)**：已解决的 Token，信号极弱。
*   **关键技术细节**：
    *   **Soft-OR 评分**：为了同时捕捉不确定性（熵）和错误（分歧），提出无参数公式：$s_t = \hat{h}_t + \hat{\delta}_t - \hat{h}_t \cdot \hat{\delta}_t$（其中 $\hat{h}_t, \hat{\delta}_t$ 为归一化值）。
    *   **Token 筛选**：在训练过程中，仅保留 Soft-OR 评分最高的 $\rho$ 比例的 Token 进行梯度更新。
    *   **理论支撑**：通过数学证明，仅靠熵会忽略 Q3 区域（过度自信的错误），而 Soft-OR 能有效恢复这一区域的信号。

*   **熵是强代理但非唯一**：保留 50% 高熵 Token 即可匹配全量训练效果，但更激进的筛选需要引入“分歧”维度。

#### 7. 优点
*   **理论与实践结合**：不仅有直观的分类法，还提供了严谨的数学证明解释“盲点”存在的原因。
*   **无参数且高效**：Soft-OR 评分不需要额外调参，且计算开销相对于反向传播几乎可以忽略。
*   **显著的资源节省**：对于工业界和学术界在有限算力下进行模型微调具有极高的实用价值。

#### 8. 不足与局限
*   **教师分布依赖**：Q3 区域的检测依赖于教师模型的输出分布，如果教师模型本身在某些位置不确定，信号质量可能受损。
*   **归一化敏感性**：Soft-OR 使用了 Batch 内的 Min-Max 归一化，可能对 Batch 中的离群 Token（Outliers）较为敏感。
*   **损失函数局限**：实验主要基于逆向 KL 散度（Reverse KL），在其他损失函数（如 Forward KL 或 JSD）下的表现尚待进一步研究。

## 摘要
大语言模型（LLMs）不断增长的上下文长度增大了键值（KV）缓存，限制了其在资源受限环境中的部署。先前的免训练 KV 缓存压缩方法通常依赖于低秩近似或标量量化，这些方法无法同时实现高压缩率和高重构保真度。我们提出了 VQKV，这是一种新颖的免训练方法，通过引入矢量量化（VQ）来获得高度压缩的 KV 表示，同时保持高模型保真度，从而仅用几个整数索引即可表示数千个浮点值。结果表明，VQKV 在 LLaMA3.1-8B 上实现了 82.8% 的压缩率，同时在 LongBench 上保留了 98.6% 的基准性能，并在相同的内存占用下实现了 4.3 倍的生成长度。

## Abstract
The growing context length of Large Language Models (LLMs) enlarges the Key-Value (KV) cache, limiting deployment in resource-limited environments. Prior training-free approaches for KV cache compression typically rely on low-rank approximation or scalar quantization, which fail to simultaneously achieve high compression ratios and high reconstruction fidelity. We propose VQKV, a novel, training-free method introducing vector quantization (VQ) to obtain highly compressed KV representations while preserving high model fidelity, allowing for the representation of thousands of floating-point values with just a few integer indices. As a result, VQKV achieves an 82.8\% compression ratio on LLaMA3.1-8B while retaining 98.6\% of the baseline performance on LongBench and enabling 4.3x longer generation length on the same memory footprint.

### 2. 论文提出的方法论：VQKV
VQKV 的核心思想是引入**矢量量化（Vector Quantization, VQ）**技术来压缩 KV 缓存。
*   **核心技术：残差简单矢量量化（RSimVQ）**：
    *   **多级残差码本**：将一个高维 KV 向量映射到多个级联的码本中。每一级量化前一级的残差，通过累加多个码本条目来逼近原始向量。
    *   **索引存储**：不再存储 16-bit 浮点数，而是存储码本条目的整数索引（Indices）。例如，用几个 10-bit 的索引代表数千个浮点维度。
    *   **SimVQ 改进**：在每个码本上增加一个额外的投影矩阵，增强对数据分布的拟合能力。
*   **算法流程**：
    1.  **离线学习**：在约 10M token 的校准数据上预训练 Key 和 Value 的专用码本（无需改动 LLM 参数）。
    2.  **预填充阶段（Prefilling）**：将生成的 KV 向量通过 RSimVQ 转换为整数索引序列并存储。
    3.  **解码阶段（Decoding）**：
        *   **按需重构**：利用自定义的 **FlashAttention 算子**，在计算注意力时才从码本中查找并求和重构 KV 向量，避免全量显存回写。
        *   **块级量化优化**：采用块级计算策略和批处理量化，减少计算开销。
    4.  **RoPE 鲁棒性**：残差设计有效地分散了旋转位置编码（RoPE）引入的维度间异构频率特征，提高了 Key 缓存的重构精度。

*   **解码延迟**：尽管进行了内核优化，但 VQ 的查表和求和操作相比原始 FP16 计算仍引入了额外的计算开销，可能导致推理首字延迟（TTFT）或吞吐量略有下降。

*   **硬件适配**：目前的高效实现依赖于 Triton 和特定架构的 GPU，在非 NVIDIA 硬件上的移植和优化尚待观察。

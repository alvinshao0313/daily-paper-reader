## 摘要
大型视觉语言模型 (VLM) 通过利用高分辨率视觉输入实现了强大的多模态理解能力，但由此产生的大量视觉 Token 造成了主要的计算瓶颈。最近的工作通过视觉 Token 压缩来缓解这一问题，通常基于显著性、多样性或两者的固定组合来压缩 Token。我们观察到，语义显著性的分布在不同样本之间存在显著差异，导致局部显著性保留与全局覆盖度之间的不同最佳权衡。这一观察表明，在所有样本中应用静态压缩策略可能是次优的。受此启发，我们提出了 PromPrune，这是一个样本自适应的视觉 Token 选择框架，由语义显著性感知预算分配和两阶段选择流水线组成。我们的方法根据每个样本的语义显著性分布，自适应地平衡局部显著性保留和全局覆盖度。通过在局部显著区域和全局多样化区域之间分配 Token 预算，我们的方法即使在高压缩率下也能保持强大的性能。在 LLaVA-NeXT-7B 上，我们的方法在保留 97.5% 原始准确率的同时，减少了 88% 的 FLOPs 和 22% 的预填充延迟。

## Abstract
Large Vision-Language Models (VLMs) achieve strong multimodal understanding capabilities by leveraging high-resolution visual inputs, but the resulting large number of visual tokens creates a major computational bottleneck. Recent work mitigates this issue through visual token compression, typically compressing tokens based on saliency, diversity, or a fixed combination of both. We observe that the distribution of semantic prominence varies substantially across samples, leading to different optimal trade-offs between local saliency preservation and global coverage. This observation suggests that applying a static compression strategy across all samples can be suboptimal. Motivated by this insight, we propose PromPrune, a sample-adaptive visual token selection framework composed of semantic prominence-aware budget allocation and a two-stage selection pipeline. Our method adaptively balances local saliency preservation and global coverage according to the semantic prominence distribution of each sample. By allocating token budgets between locally salient regions and globally diverse regions, our method maintains strong performance even under high compression ratios. On LLaVA-NeXT-7B, our approach reduces FLOPs by 88% and prefill latency by 22% while preserving 97.5% of the original accuracy.

### 1. 核心问题与研究动机
*   **核心问题**：现代大视觉语言模型（VLM）为了提升理解能力，通常采用高分辨率输入，这导致视觉 Token 数量激增，造成了巨大的计算开销、预填充延迟和 KV 缓存压力。
*   **研究动机**：现有的 Token 压缩方法（如剪枝或合并）通常采用**静态策略**，即要么只关注局部显著性（Saliency），要么只关注全局覆盖度（Coverage），或者以固定比例组合两者。作者发现，不同图像的语义分布差异巨大：有的任务（如 SQA）依赖局部细节，而有的任务（如 POPE）需要全局上下文。因此，**样本无关的静态压缩策略是次优的**，需要一种能根据图像内容自适应调整压缩侧重点的方法。

### 2. 方法论：PromPrune 框架
PromPrune 的核心思想是根据样本的“语义显著性分布”动态分配 Token 预算。
*   **语义显著性度量（奇异值谱熵）**：作者提出使用视觉 Token 特征矩阵的**谱熵（Spectral Entropy）**来衡量语义分布。通过对 Token 矩阵进行奇异值分解（SVD），计算归一化奇异值的熵。高谱熵意味着语义分布广泛（需更多覆盖度），低谱熵意味着语义集中在少数区域（需更多显著性）。
*   **自适应预算分配**：利用一个 Sigmoid 映射函数，将计算出的谱熵转化为两个预算指标：
    1.  **$T_{sal}$（显著性预算）**：用于保留局部核心语义。
    2.  **$T_{cov}$（覆盖度预算）**：用于保留全局多样性背景。
*   **两阶段选择流水线**：
    *   **第一阶段（显著性选择）**：根据视觉编码器的注意力分数（CLS-to-token attention），选取前 $T_{sal}$ 个最显著的 Token。
    *   **第二阶段（覆盖度补充）**：从剩余 Token 中，利用**行列式点过程（DPP）**算法选取 $T_{cov}$ 个 Token。DPP 能够最大化所选 Token 集合的多样性，确保全局覆盖。
*   **即插即用**：该方法不需要重新训练模型或修改 LLM 架构。

### 6. 主要结论与发现
*   **性能领先**：在极高压缩率下（如仅保留 5% 的 Token），PromPrune 的表现显著优于所有静态压缩方法。例如在 LLaVA-1.5 上保留 32 个 Token 时，相对原始性能仍保持 93.5%。
*   **自适应的必要性**：实验证实，谱熵确实能反映图像的语义集中度，且基于此进行的动态预算分配能有效平衡“看清物体”和“看全场景”的需求。
*   **计算开销可控**：虽然引入了 SVD 和 DPP 计算，但相比 LLM 庞大的推理开销，这些额外延迟（约 14ms）几乎可以忽略不计。

### 7. 优点
*   **样本自适应**：首次深入探讨并解决了视觉 Token 压缩中显著性与覆盖度的动态权衡问题。
*   **理论支撑强**：引入谱熵作为度量工具，比单纯的注意力权重更具鲁棒性。
*   **通用性好**：支持多种 VLM 架构，且不需要昂贵的重训练过程。
*   **极端压缩下的稳定性**：在 Token 预算极低的情况下，性能下降曲线比同类方法更平缓。

### 8. 不足与局限
*   **多图/切片处理**：在处理 LLaVA-NeXT 这种将图像切分为多个局部窗口的架构时，目前的预算分配主要基于全局图，可能未完全优化每个局部切片的 Token 分配。
*   **超参数依赖**：Sigmoid 映射中的偏置参数 $\mu$ 需要根据不同视觉编码器的特征分布进行一次性的轻量级预估（虽然论文证明了其具有一定的鲁棒性）。
*   **应用限制**：主要针对静态图像理解，尚未在长视频理解等更极端的长序列场景中进行大规模验证。

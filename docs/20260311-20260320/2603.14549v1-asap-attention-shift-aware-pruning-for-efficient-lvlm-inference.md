## 摘要
尽管大型视觉语言模型（LVLMs）展现出卓越的多模态能力，但处理高分辨率视觉 token 的二次方计算成本仍然是一个关键瓶颈。虽然近期的 token 减少策略试图加速推理，但这些方法未能充分利用注意力值，且未能解决 token 冗余问题。更关键的是，它们忽视了 LVLMs 中固有的“注意力偏移”现象，该现象会导致 token 注意力分数产生偏差。在这项工作中，我们提出了 ASAP，这是一种新型的、无需训练且兼容 KV-Cache 的剪枝方案，全面解决了上述局限性。首先，我们利用动态双向软注意力掩码来缓解注意力偏移，确保选择真正具有信息量的 token，而非基于朴素注意力的选择。其次，我们认为 token 集合中的高语义冗余会降低性能。因此，我们引入了一个加权软合并组件来合并语义相似的 token，仅为后续层保留特征最密集的视觉补丁。ASAP 实现了几乎无损的视觉上下文压缩，在保留 LLaVA-NeXT-7B 原始性能 99.02% 的同时，大幅削减了约 80% 的计算 FLOPs。

## Abstract
While Large Vision-Language Models (LVLMs) demonstrate exceptional multi-modal capabilities, the quadratic computational cost of processing high-resolution visual tokens remains a critical bottleneck. Though recent token reduction strategies attempt to accelerate inference, such methods inadequately exploit attention values and fail to address token redundancy. More critically, they overlook the ``attention shift'' phenomenon inherent in LVLMs, which skews token attention scores. In this work, we propose ASAP, a novel training-free, KV-Cache-compatible pruning recipe that comprehensively addresses these limitations. First, we mitigate the attention shift by utilizing a dynamic bidirectional soft attention mask, ensuring the selection of genuinely informative tokens rather than naive attention-based selection. Second, we posit that high semantic redundancy within the token set degrades performance. We therefore introduce a weighted soft merging component that merges semantically similar tokens, preserving only the most feature-dense visual patches for subsequent layers. ASAP achieves virtually lossless compression of visual context, retaining 99.02% of the original LLaVA-NeXT-7B performance while aggressively slashing computational FLOPs by ~80%.

### 论文总结：ASAP（注意力偏移感知剪枝）

#### 1. 核心问题与研究动机
*   **核心问题**：大型视觉语言模型（LVLM）在处理高分辨率图像时会产生海量的视觉 Token（如 LLaVA-NeXT 产生约 2304 个），导致 Transformer 的计算复杂度随序列长度呈二次方增长，严重制约推理效率。
*   **研究动机**：
    *   **注意力偏移（Attention Shift）**：作者发现由于旋转位置编码（RoPE）的距离衰减特性，模型会下意识地给序列后端的 Token（图像底部）分配更高权重，而忽略前端（图像顶部）的重要信息。
    *   **语义冗余**：图像中存在大量重复背景（如天空、海洋），现有剪枝方法未能有效识别并压缩这些冗余。
    *   **现有方法缺陷**：现有的 Token 减少技术要么忽略了文本查询的引导，要么在剪枝时未考虑 RoPE 带来的结构性偏差。

#### 2. 方法论：ASAP 框架
ASAP 是一种**无需训练（Training-free）**且**兼容 KV 缓存**的剪枝方案，包含三个核心组件：
*   **显著性引导的双向注意力掩码（SG-BiMask）**：
    *   **核心思想**：打破传统的因果掩码（Causal Mask）限制，允许视觉 Token 之间进行双向信息流动。
    *   **技术细节**：利用预 Softmax 的对齐矩阵计算 Token 的结构显著性，并引入一个动态的软掩码（Soft Mask）。通过在对数空间应用显著性相关的惩罚因子，让重要的早期视觉 Token 能够“看到”后续 Token，从而抵消 RoPE 的距离衰减效应。
*   **显著性加权合并（Salience-Weighted Consolidation）**：
    *   **核心思想**：在保留关键 Token 的基础上，通过特征相似度合并冗余 Token。
    *   **技术细节**：计算 Token 间的余弦相似度，并根据显著性得分进行加权融合，确保融合后的特征向最具结构意义的锚点靠拢，避免特征稀释。
*   **预算重新分配（Budget Reallocation）**：
    *   **核心思想**：利用合并操作释放出的 Token 槽位，从之前的剪枝池中“打捞”次优但仍具显著性的 Token，以最大化固定预算下的信息密度。

*   **推理开销**：虽然减少了 FLOPs，但动态 Token 选择和合并操作引入了微小的推理开销（约 2ms 的延迟增加），在极度追求实时性的场景下需权衡。

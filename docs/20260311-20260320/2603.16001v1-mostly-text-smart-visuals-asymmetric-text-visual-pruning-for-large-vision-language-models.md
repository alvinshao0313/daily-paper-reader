## 摘要
网络剪枝是实现轻量化大视觉语言模型（LVLMs）的一种有效技术，它主要将权重和激活值都纳入重要性度量中。然而，现有的工作通常以统一的方式处理来自不同模态的校准数据，忽略了模态特定的行为。这提出了一个关键挑战：如何处理文本和视觉 token 的差异化行为，以实现 LVLMs 的精确剪枝。为此，我们通过解耦相应的权重，系统地研究了视觉和文本 token 对剪枝操作的敏感性，结果表明：(i) 文本路径应通过文本 token 进行校准，因为它比视觉路径表现出更高的敏感性；(ii) 视觉路径表现出高度冗余，甚至允许 50% 的稀疏度。受这些见解的启发，我们为 LVLMs 提出了一种简单而有效的不对称文本-视觉权重剪枝方法，称为 ATV-Pruning，该方法通过从文本和视觉路径中选择信息丰富的 token 来建立精确权重剪枝的重要性度量。具体而言，ATV-Pruning 整合了两项主要创新：首先，通过利用所有文本 token 和视觉 token 的子集自适应地构建校准池；其次，我们设计了一种层自适应选择策略来产生重要的视觉 token。最后，在标准多模态基准测试上的广泛实验验证了我们的 ATV-Pruning 优于现有最先进的方法。

## Abstract
Network pruning is an effective technique for enabling lightweight Large Vision-Language Models (LVLMs), which primarily incorporates both weights and activations into the importance metric. However, existing efforts typically process calibration data from different modalities in a unified manner, overlooking modality-specific behaviors. This raises a critical challenge: how to address the divergent behaviors of textual and visual tokens for accurate pruning of LVLMs. To this end, we systematically investigate the sensitivity of visual and textual tokens to the pruning operation by decoupling their corresponding weights, revealing that: (i) the textual pathway should be calibrated via text tokens, since it exhibits higher sensitivity than the visual pathway; (ii) the visual pathway exhibits high redundancy, permitting even 50% sparsity. Motivated by these insights, we propose a simple yet effective Asymmetric Text-Visual Weight Pruning method for LVLMs, dubbed ATV-Pruning, which establishes the importance metric for accurate weight pruning by selecting the informative tokens from both textual and visual pathways. Specifically, ATV-Pruning integrates two primary innovations: first, a calibration pool is adaptively constructed by drawing on all textual tokens and a subset of visual tokens; second, we devise a layer-adaptive selection strategy to yield important visual tokens. Finally, extensive experiments across standard multimodal benchmarks verify the superiority of our ATV-Pruning over state-of-the-art methods.

### 1. 核心问题与整体含义（研究动机和背景）
随着大视觉语言模型（LVLM）规模的不断扩大，其在资源受限设备上的部署面临巨大挑战。**网络剪枝**是减轻模型负担的关键技术，但现有方法（如 Wanda、SparseGPT）通常将文本和视觉模态的 Token 统一处理，忽略了不同模态在剪枝敏感度上的**异质性**。
论文通过实验发现：
*   **文本路径高度敏感**：必须使用文本 Token 进行校准，否则性能会大幅下降。
*   **视觉路径高度冗余**：即使在 60% 的稀疏度下，视觉路径依然表现出极强的鲁棒性。
基于此，论文提出了 **ATV-Pruning**，旨在通过“非对称”地处理文本和视觉信息，构建更高效的剪枝校准池。

### 2. 方法论：核心思想与技术细节
ATV-Pruning 的核心思想是：**以文本为锚点，精简视觉校准信息**。
*   **非对称校准池构建**：不同于传统方法使用全部 Token，ATV-Pruning 的校准池包含**全部文本 Token** 和一个**经过筛选的视觉 Token 子集**。
*   **视觉漂移（Visual Drift）显著性度量**：
    *   定义视觉 Token 在经过一个 Transformer 块前后的表示变化（余弦距离）为“视觉漂移”。
    *   漂移越大，说明该 Token 在该层参与的计算越活跃，重要性越高。
*   **层自适应选择策略**：
    *   根据每一层平均的视觉漂移程度，动态分配视觉 Token 的配额（Budget）。
    *   在视觉处理活跃的层保留更多视觉 Token，在稳定的层则减少保留。
*   **剪枝流程**：利用构建好的非对称校准池计算权重的重要性得分（结合权重幅值与激活值范数），最后对 LLM 主干网络进行非结构化或半结构化剪枝。

### 3. 实验设计
*   **模型对象**：LLaVA-NeXT (8B), Qwen2-VL (7B), LLaVA-OneVision (7B), Qwen2.5-VL (7B)。

*   **对比方法**：
    *   **SparseGPT**：基于 Hessian 矩阵的权重更新剪枝。
    *   **Wanda**：基于激活值的快速剪枝。
    *   **TAMP**：现有的多模态自适应剪枝 SOTA 方法。
*   **稀疏度设置**：主要测试 50% 和 60% 的非结构化稀疏度，以及 2:4 和 4:8 的半结构化稀疏度。

*   **硬件**：所有实验均在**单张 NVIDIA A100 GPU** 上完成。

### 5. 实验数量与充分性
论文进行了非常充分的实验验证：

*   **公平性**：所有方法使用相同的校准样本（来自 ShareGPT4V 的 128 个图文对），确保了对比的客观性。

### 6. 主要结论与发现
*   **非对称性是关键**：文本信息对维持模型逻辑和语言能力至关重要，而视觉信息在校准时存在大量冗余。
*   **性能领先**：在 60% 稀疏度下，ATV-Pruning 在多个基准上的表现显著优于 Wanda 和 TAMP，甚至在不进行权重更新的情况下超越了计算密集的 SparseGPT。
*   **视觉漂移的有效性**：实验证明基于表示变化的“视觉漂移”比基于注意力或随机选择的策略更能捕捉关键视觉特征。

### 7. 优点
*   **动机明确且直观**：通过解耦实验揭示了模态间的敏感度差异，为非对称剪枝提供了坚实的理论支撑。
*   **高效且易于集成**：作为一种 Post-training Pruning 方法，它不需要重新训练，且计算开销极低，适合快速部署。
*   **通用性强**：在不同家族的模型（LLaVA 系列和 Qwen 系列）上均有效。

### 8. 不足与局限
*   **超参数依赖**：虽然 $\alpha=1$ 在多数实验中表现良好，但针对极高性能要求的场景，可能仍需针对特定模型微调该缩放因子。
*   **模态扩展性**：目前仅针对图像和文本，尚未探讨在包含音频、视频或深度信息等更多模态的大型多模态模型（LMM）中的表现。
*   **剪枝粒度**：主要集中在 LLM 主干网络，未深入探讨视觉编码器（Vision Tower）与连接器（Connector）的同步剪枝优化。

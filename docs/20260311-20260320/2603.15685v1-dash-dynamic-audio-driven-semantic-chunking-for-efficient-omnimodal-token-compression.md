## 摘要
全模态大语言模型（OmniLLMs）共同处理音频和视觉流，但由此产生的长多模态 Token 序列使得推理成本极高。现有的压缩方法通常依赖于固定窗口划分和基于注意力的剪枝，这忽略了视听信号的分段语义结构，并且在激进的 Token 削减下变得脆弱。我们提出了动态音频驱动语义分块（DASH），这是一个无需训练的框架，旨在使 Token 压缩与语义结构保持一致。DASH 将音频嵌入视为语义锚点，并通过余弦相似度不连续性检测候选边界，从而诱导出动态的、变长的分段，这些分段近似于序列底层分段相干的组织结构。这些边界被投影到视频 Token 上，以建立显式的跨模态分割。在每个分段内，Token 的保留由一个三信号重要性估计器决定，该估计器融合了结构边界线索、表征独特性和基于注意力的显著性，从而减轻了仅基于注意力的选择所带来的稀疏性偏差。这种结构感知的分配在减少冗余区域的同时，保留了对转换至关重要的 Token。在 AVUT、VideoMME 和 WorldSense 上的广泛实验表明，与之前的方法相比，DASH 在实现更高压缩率的同时保持了卓越的准确性。代码可在以下网址获取：https://github.com/laychou666/DASH。

## Abstract
Omnimodal large language models (OmniLLMs) jointly process audio and visual streams, but the resulting long multimodal token sequences make inference prohibitively expensive. Existing compression methods typically rely on fixed window partitioning and attention-based pruning, which overlook the piecewise semantic structure of audio-visual signals and become fragile under aggressive token reduction. We propose Dynamic Audio-driven Semantic cHunking (DASH), a training-free framework that aligns token compression with semantic structure. DASH treats audio embeddings as a semantic anchor and detects boundary candidates via cosine-similarity discontinuities, inducing dynamic, variable-length segments that approximate the underlying piecewise-coherent organization of the sequence. These boundaries are projected onto video tokens to establish explicit cross-modal segmentation. Within each segment, token retention is determined by a tri-signal importance estimator that fuses structural boundary cues, representational distinctiveness, and attention-based salience, mitigating the sparsity bias of attention-only selection. This structure-aware allocation preserves transition-critical tokens while reducing redundant regions. Extensive experiments on AVUT, VideoMME, and WorldSense demonstrate that DASH maintains superior accuracy while achieving higher compression ratios compared to prior methods. Code is available at: https://github.com/laychou666/DASH.

#### 1. 核心问题与整体含义（研究动机和背景）
全模态大语言模型（OmniLLMs）在处理视频时，由于需要同时处理音频和视觉流，会产生极长的 Token 序列，导致推理时的计算开销（内存和延迟）呈平方级增长。
*   **现有挑战**：目前的 Token 压缩方法（如剪枝或合并）通常采用**固定窗口划分**和**单一的注意力机制引导**。这种做法忽略了音视频信号内在的语义结构（如语音停顿、场景切换），且在极高压缩率下容易丢失关键的转折信息，导致模型理解能力大幅下降。
*   **研究动机**：论文认为音频流是天然的“语义锚点”，其嵌入向量在语义转折处（如说话停顿、主题切换）表现出明显的分布不连续性。利用音频引导视频分段，可以实现更符合人类感知的结构化压缩。

#### 2. 方法论：核心思想与关键技术
DASH 是一个**无需训练（Training-free）**的即插即用框架，核心流程分为四个阶段：
*   **动态语义分块（音频端）**：通过计算相邻音频 Token 嵌入向量的**余弦相似度**来检测边界。当相似度低于阈值且满足最小块长度限制时，判定为语义边界，从而产生变长的动态分段。
*   **跨模态边界投影（视频端）**：利用时间比例映射（Temporal Ratio Mapping），将音频检测到的语义边界投影到视频 Token 索引上，实现音视频在语义层面的显式对齐。
*   **三信号融合 Token 评分**：为了克服单一注意力机制的稀疏性偏差，DASH 融合了三个维度的信号来评估 Token 重要性：
    1.  **边界概率**：优先保留处于语义转折点的 Token。
    2.  **多尺度高斯独特性**：利用密度峰值聚类原理，识别表征上最具独特性的 Token。
    3.  **注意力分数**：保留模型感知到的显著性 Token。
*   **边界感知视频压缩**：根据音频段的保留率动态调整对应视频段的压缩比例，并对边界帧进行额外保护，确保叙事连续性。

#### 6. 主要结论与发现
*   **高效压缩**：DASH 在 **25%** 的 Token 保留率下，准确度即可达到或超过前人方法在 35% 保留率下的表现。

*   **结构感知的重要性**：实验证明，遵循语义边界而非固定窗口进行压缩，能显著减少激进剪枝下的信息损失，尤其是在需要跨模态推理的复杂任务中。

#### 7. 优点与亮点
*   **创新视角**：首次提出将音频作为全模态压缩的语义锚点，解决了视频 Token 冗余且结构模糊的问题。

*   **实用性强**：无需重新训练模型，计算开销极小，易于集成到现有的 OmniLLM 推理流水线中。

*   **音频依赖性**：该方法高度依赖音频信号的质量。在静音视频或背景噪音极大的极端场景下，音频作为语义锚点的可靠性可能会下降（虽然文中提到有退化处理机制）。

*   **应用限制**：目前主要针对双模态（音+视）输入，对于更多模态（如传感器数据）的扩展性尚未探讨。

## 摘要
上下文稀疏性是降低大语言模型（LLM）推理过程中计算复杂度的方法之一。现有的基于上下文稀疏性且精度损失极小的 LLM 推理加速技术通常需要训练稀疏模式预测器。本文提出了一种用于加速 LLM 中基于 ReGLU 的前馈网络（FFN）推理的框架。该框架提供了一种快速、无需训练的方法，利用门控投影矩阵的截断感知奇异值分解（SVD）来构建稀疏模式预测器，并结合了阈值校准算法，以及支持在 CUDA 和 CANN 设备上进行条件计算的推理执行器。在三个 FFN 平均激活稀疏度为 90% 的稀疏 LLM 上进行的实验表明，在涉及复杂数学和代码生成的任务中，端到端解码时间缩短了高达 1.8 倍，同时基准测试分数下降不到 1%。这项工作推进了 LLM 在边缘设备上的部署。

## Abstract
Contextual sparsity is one of the approaches used to reduce computational complexity in the inference process of large language models (LLMs). Existing techniques for efficient LLM inference acceleration based on contextual sparsity with minimal accuracy degradation require training sparse pattern predictors. This paper presents a framework for accelerating inference of ReGLU-based feed-forward networks (FFNs) within LLMs. The proposed framework provides a fast, training-free method for building sparse pattern predictors using truncation-aware singular value decomposition (SVD) of the gate projection matrix, along with a threshold calibration algorithm, and inference executors supporting conditional computation on CUDA and CANN devices. Experiments on three sparse LLMs with an average activation sparsity level of 90% in the FFNs demonstrate up to a 1.8x reduction in end-to-end decoding time while maintaining less than 1% degradation in benchmark scores on tasks involving complex math and code generation. This work advances the deployment of LLMs on edge devices.

以下是对该论文的结构化总结：

### 1. 核心问题与研究动机
*   **核心问题**：LLM 推理（尤其是边缘设备部署）面临巨大的计算和内存带宽挑战。基于 ReGLU 激活函数的 LLM 存在“上下文稀疏性”现象，即在特定输入下，FFN 层中只有极少数神经元被激活（通常少于 10%）。
*   **研究动机**：现有的稀疏性加速方案（如 Deja Vu, PowerInfer）通常需要复杂的**有监督训练**来构建稀疏模式预测器，这增加了部署成本和模型迁移的难度。本文旨在提出一种**免训练（Training-free）**、基于线性代数分解的高效预测方案。

### 2. 方法论
核心思想是利用奇异值分解（SVD）对 FFN 的门控投影矩阵进行低秩近似，从而构建轻量级的预测器。
*   **SVD 预测器构建**：将门控矩阵 $W_{gate}$ 近似分解为两个低秩矩阵 $A$ 和 $B$。预测器形式为 $P(x) = H(ABx + b)$，其中 $H$ 是阶跃函数。
*   **截断感知数据白化（Data Whitening）**：为了降低 SVD 截断带来的 Frobenius 范数误差，引入了基于校准数据的协方差矩阵进行白化处理，使预测器更贴合实际输入分布。
*   **贪心偏置校准算法（Greedy Bias Calibration）**：直接使用 SVD 会导致严重的精度下降。论文提出一种无需梯度的贪心算法，为每个神经元计算最优偏置（阈值）$b$。该算法在给定稀疏度预算下，优先保留对输出影响最大的神经元。
*   **顺序执行流水线（Sequential Pipeline）**：不同于传统的并行计算 Gate 和 Up 投影，该方法先计算 Gate 投影，利用其结果进一步过滤掉预测器产生的“假阳性”神经元，从而在 Up 和 Down 投影阶段实现更高的实际稀疏度。
*   **硬件执行器**：开发了针对 NVIDIA CUDA（自定义内核）和华为昇腾 CANN（基于 PyTorch 高级 API）的稀疏 FFN 执行器。

### 3. 实验设计
*   **实验模型**：选取了三个具有代表性的稀疏 7B 模型：ProSparse-LLaMA2-7B、SparseQwen2-7B 和 TurboSparse-Mistral-Instruct。
*   **基准测试（Benchmarks）**：涵盖了数学推理（GSM8K）、代码生成（HumanEval, MBPP）、科学常识（ARC-E/C）、通用问答（TriviaQA）、复杂推理（BBH）和中文多任务理解（CMMLU）。
*   **对比方法**：
    *   **有训练方法**：Deja Vu, PowerInfer。
    *   **免训练方法**：GRIFFIN。
    *   **基准**：全密集计算（Dense baseline）。

### 4. 资源与算力
*   **GPU 平台**：使用单张 **NVIDIA RTX 3090** 进行精度和性能评估（由莫斯科国立大学实验室完成）。
*   **NPU 平台**：使用单张 **昇腾 Ascend 310P3** 进行性能评估（由华为团队完成）。
*   **训练/校准开销**：该方法无需反向传播训练。离线校准阶段仅需约 **112 个样本（约 20,000 个 token）**，在双精度下进行 SVD 分解和贪心校准，耗时极短。

### 5. 实验数量与充分性
*   **实验规模**：在 3 个模型上进行了 8 类主流基准测试，涵盖了从 0.4 到 0.7 的不同预测稀疏度设置。
*   **消融实验**：详细对比了“纯 SVD”、“加入白化”、“加入偏置校准”以及“并行 vs 顺序流水线”的效果。
*   **客观性**：实验对比了预测器的 ROC-AUC 曲线，证明了 SVD 预测器在未见数据上的泛化能力优于某些需要训练的方法。实验设计较为全面，考虑了不同硬件架构的适配性。

### 6. 主要结论与发现
*   **加速效果**：在保持极高精度的前提下，端到端（E2E）解码速度提升高达 **1.8 倍**。
*   **精度保持**：在 FFN 稀疏度达到 90% 时，数学和代码任务的精度损失控制在 **1% 以内**。
*   **预测质量**：SVD 预测器的分离能力（AUROC）在深层网络中表现尤为突出，且由于其基于权重结构而非纯数据驱动，表现出更好的鲁棒性。
*   **硬件友好性**：在昇腾平台上，仅通过 PyTorch API 即可获得显著加速，证明了该方案对国产算力平台的适配潜力。

### 7. 优点与亮点
*   **免训练部署**：极大地简化了稀疏 LLM 的优化流程，仅需少量数据即可完成校准。
*   **理论支撑**：论文提供了关于预测器误差上界的理论分析，并证明了贪心校准算法在特定条件下的全局最优性。
*   **顺序流水线优化**：巧妙利用 ReGLU 的特性，通过二次过滤降低了假阳性率，平衡了速度与精度。

### 8. 不足与局限
*   **架构限制**：该方法目前主要针对 **ReGLU 或 ReLU** 激活函数的模型。对于使用 SiLU 等平滑激活函数的模型，直接应用会导致明显的精度下降（如文中对 Qwen2-Instruct 的测试所示）。
*   **Batch Size 限制**：实验主要针对 **Batch Size = 1** 的场景（边缘设备典型场景）。在高性能服务器的大 Batch 场景下，由于神经元激活模式的并集会迅速扩大，稀疏性带来的增益会显著减小。
*   **Prefill 阶段**：该方法目前仅加速解码（Decoding）阶段，未解决 Prefill 阶段的稀疏化计算问题。

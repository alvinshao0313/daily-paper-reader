## 摘要
上下文稀疏性是降低大语言模型（LLM）推理过程中计算复杂度的方法之一。现有的基于上下文稀疏性且精度损失极小的 LLM 推理加速技术通常需要训练稀疏模式预测器。本文提出了一种用于加速 LLM 中基于 ReGLU 的前馈网络（FFN）推理的框架。该框架提供了一种快速、无需训练的方法，通过对门控投影矩阵进行截断感知的奇异值分解（SVD）来构建稀疏模式预测器，并结合了阈值校准算法，以及支持在 CUDA 和 CANN 设备上进行条件计算的推理执行器。在三个 FFN 平均激活稀疏度为 90% 的稀疏 LLM 上进行的实验表明，在涉及复杂数学和代码生成的任务中，端到端解码时间缩短了高达 1.8 倍，同时基准测试分数的下降控制在 1% 以内。这项工作推进了 LLM 在边缘设备上的部署。

## Abstract
Contextual sparsity is one of the approaches used to reduce computational complexity in the inference process of large language models (LLMs). Existing techniques for efficient LLM inference acceleration based on contextual sparsity with minimal accuracy degradation require training sparse pattern predictors. This paper presents a framework for accelerating inference of ReGLU-based feed-forward networks (FFNs) within LLMs. The proposed framework provides a fast, training-free method for building sparse pattern predictors using truncation-aware singular value decomposition (SVD) of the gate projection matrix, along with a threshold calibration algorithm, and inference executors supporting conditional computation on CUDA and CANN devices. Experiments on three sparse LLMs with an average activation sparsity level of 90% in the FFNs demonstrate up to a 1.8x reduction in end-to-end decoding time while maintaining less than 1% degradation in benchmark scores on tasks involving complex math and code generation. This work advances the deployment of LLMs on edge devices.

### 1. 核心问题与研究动机
*   **核心问题**：大语言模型（LLM）推理计算量巨大，尤其在边缘设备上部署挑战极高。
*   **研究背景**：基于 ReGLU 激活函数的大模型存在“上下文稀疏性”（Contextual Sparsity），即在 FFN 层中只有极少数神经元会被激活。
*   **动机**：现有的稀疏性预测方法（如 Deja Vu）通常需要复杂的模型训练来学习预测掩码。本文旨在提出一种**无需训练（Training-free）**、基于奇异值分解（SVD）的轻量级预测器，以实现低延迟、高精度的推理加速。

### 2. 方法论
该框架分为**离线校准**和**在线推理**两个阶段：
*   **核心思想**：利用门控投影矩阵 $W_{gate}$ 的低秩 SVD 近似作为稀疏性预测器。
*   **关键技术细节**：
    *   **截断感知 SVD 与数据白化**：通过校准数据集的协方差矩阵进行数据白化（Data Whitening），最小化 Frobenius 范数误差，提升低秩近似的准确性。
    *   **贪心偏置校准算法**：引入神经元特异性的偏置项（Bias）作为阈值。使用贪心算法在给定稀疏度预算下，优先保留对模型输出影响最大的神经元，补偿 SVD 截断带来的分布偏移。
    *   **顺序执行流水线**：不同于传统的并行计算，该方法先计算门控投影，利用 ReLU 的特性剔除假阳性神经元，进一步提高后续 Up 和 Down 投影的实际稀疏度。
    *   **硬件执行器**：开发了支持 CUDA（NVIDIA）和 CANN（华为昇腾）的自定义稀疏算子，通过共享内存规约等技术保证计算的确定性和高效性。

### 6. 主要结论与发现
*   **加速效果**：在 FFN 层平均激活稀疏度达到 90% 的情况下，实现了高达 **1.8 倍** 的端到端解码加速。

### 7. 优点（亮点）
*   **免训练**：无需昂贵的反向传播训练，仅需少量数据校准，部署极快。
*   **理论支撑**：为 SVD 预测器的误差界限提供了确定性的理论分析。
*   **通用性**：支持多种 ReGLU 变体模型，并在两种主流 AI 硬件架构上均得到验证。
*   **高效性**：通过顺序流水线设计，实际执行的稀疏度往往高于预测稀疏度。

### 8. 不足与局限
*   **激活函数限制**：目前仅适用于基于 ReLU 或 ReGLU 的稀疏模型。对于使用 SiLU 等平滑激活函数且未经过稀疏化微调的模型，效果较差（如文中提到在 SiLU 版 Qwen2 上精度下降明显）。
*   **Batch Size 限制**：主要针对 Batch Size = 1 的推理场景（边缘设备常见），在处理大 Batch 推理时，由于不同序列的稀疏模式重叠，加速效果会显著减弱。
*   **Prefill 阶段**：该方法主要加速 Decoding（生成）阶段，由于 Prefill 阶段通常是计算密集型且难以利用动态稀疏性，目前尚未覆盖。

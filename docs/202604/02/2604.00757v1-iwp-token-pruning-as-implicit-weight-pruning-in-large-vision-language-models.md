## 摘要
大视觉语言模型在图像和视频理解任务中表现出卓越的性能，但其计算成本随视觉 Token 数量的增加而迅速增长。现有的 Token 剪枝方法通过经验性方法缓解了这一问题，但忽视了注意力的内部机制。在本文中，我们提出了一种基于注意力对偶形式视角的新型免训练 Token 剪枝框架。我们将注意力重新表述为一个隐式线性层，其权重矩阵是秩为 1 的外积之和，每个外积由单个 Token 的键值对生成。因此，Token 剪枝简化为选择这些秩为 1 更新的最优子集，以最好地近似原始对偶权重矩阵。将这一视角扩展到 LVLM 中的标准 softmax 注意力，我们推导出一种衡量 Token 信息量和信息重复度的新型指标。为了利用所提指标高效地选择子集，我们引入了渐进式分块最大边际相关性（Progressive Chunked Maximal Marginal Relevance）。大量的实验表明，我们的方法在性能和效率之间实现了更好的权衡，同时为现有的剪枝方法提供了另一种视角。

## Abstract
Large Vision Language Models show impressive performance across image and video understanding tasks, yet their computational cost grows rapidly with the number of visual tokens. Existing token pruning methods mitigate this issue through empirical approaches while overlooking the internal mechanism of attention. In this paper, we propose a novel training free token pruning framework grounded in the dual form perspective of attention. We reformulate attention as an implicit linear layer whose weight matrix is the sum of rank 1 outer products, each generated by a single token's key value pair. Token pruning thus reduces to selecting an optimal subset of these rank 1 updates that best approximates the original dual weight matrix. Extending this perspective to standard softmax attention in LVLMs, we derive a novel metric quantifying both a token's information magnitude and information duplication. To efficiently select the subset with the proposed metric, we introduce Progressive Chunked Maximal Marginal Relevance. Extensive experiments demonstrate that our method achieves a better trade off between performance and efficiency, while providing another perspective on existing pruning approaches.

#### 1. 核心问题与整体含义（研究动机和背景）
随着大视觉语言模型（LVLM）向高分辨率图像和长视频理解演进，视觉 Token 的数量呈爆炸式增长。由于自注意力机制的计算复杂度随序列长度呈平方级增长，这带来了巨大的计算和内存开销。现有的 Token 剪枝方法大多基于经验（如直接使用注意力分数或简单的多样性约束），缺乏对注意力机制内部运作逻辑的深入理解。本论文旨在从**注意力的对偶形式（Dual Form）**出发，建立一个有理论支撑的、免训练的 Token 剪枝框架，将 Token 剪枝视为对注意力隐式权重矩阵的近似过程。

#### 2. 方法论：核心思想与关键技术
*   **核心思想**：将标准注意力机制重新表述为隐式线性变换。在线性注意力中，输出可视为 Query 与一个隐式权重矩阵 $W$ 的乘积，而 $W$ 是所有 Token 的 Key-Value 对生成的**秩-1（rank-1）外积**之和。因此，剪枝的目标是选择一个 Token 子集，使其生成的秩-1 更新之和能最大程度地保留原始权重矩阵的信息。
*   **关键技术细节**：
    *   **指标推导**：将上述视角扩展到标准 Softmax 注意力，推导出两个核心指标：
        1.  **信息强度（Information Magnitude）**：衡量单个 Token 对权重矩阵更新的贡献大小，结合了 Key-Query 的对齐程度和 Value 的范数。
        2.  **信息重复度（Information Duplication）**：衡量不同 Token 诱导的更新之间的相似性（基于核函数空间中的 Key 相似度和 Value 向量的余弦相似度）。
    *   **PC-MMR 算法**：为了高效选择子集，提出了**渐进式分块最大边际相关性（Progressive Chunked MMR）**算法。该算法不再进行耗时的逐个 Token 选择，而是以分块（Chunk）形式迭代，在选择高信息强度 Token 的同时，通过惩罚项抑制与已选集合重复度高的 Token，并随迭代进度动态增加块大小以加速推理。

*   **性能领先**：在所有测试的 Token 预算下，IWP 均优于现有方法。特别是在极低预算（11.1%）下，其性能保持能力显著强于基于纯注意力分数的方法。

*   **理论深刻**：跳出了经验性剪枝的范畴，从数学对偶性角度重新定义了 Token 重要性。

*   **平衡性好**：通过 PC-MMR 算法巧妙平衡了“重要性”与“多样性”，有效缓解了剪枝过程中的信息坍缩。

*   **模态局限**：目前的实验主要集中在视觉模态，虽然理论上模态无关，但在音频或纯文本长序列上的效果尚未验证。

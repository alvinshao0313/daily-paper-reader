## 摘要
在多模态大语言模型（MLLM）中，视觉 Token 的激增显著增加了推理时间和计算开销，使其在实时或资源受限的应用中变得不切实际。视觉 Token 剪枝是一种通过移除冗余视觉 Token 来降低 MLLM 推理成本的有前景的策略。现有研究通常假设所有注意力头对视觉解释的贡献是均等的。然而，我们的研究表明，不同的注意力头可能会捕捉到不同的视觉语义，并在视觉处理中发挥截然不同的作用。基于这一观察，我们提出了 HAWK，一种注意力头重要性感知的视觉 Token 剪枝方法，该方法通过感知注意力头在视觉任务中不同的重要性，以最大限度地保留关键 Token。通过利用注意力头重要性权重和文本引导的注意力来评估视觉 Token 的显著性，HAWK 有效地保留了与任务相关的视觉 Token，同时移除了冗余 Token。所提出的 HAWK 完全无需训练，且可以无缝应用于各种 MLLM。在多个主流视觉语言基准测试上的广泛实验表明，HAWK 达到了最先进的准确率。当应用于 Qwen2.5-VL 时，HAWK 在剪枝 80.2% 的视觉 Token 后，仍保留了原始准确率的 96.0%。此外，它将端到端延迟降低至原始水平的 74.4%，并进一步减少了所测试模型的 GPU 显存占用。代码可在 https://github.com/peppery77/HAWK.git 获取。

## Abstract
In multimodal large language models (MLLMs), the surge of visual tokens significantly increases the inference time and computational overhead, making them impractical for real-time or resource-constrained applications. Visual token pruning is a promising strategy for reducing the cost of MLLM inference by removing redundant visual tokens. Existing research usually assumes that all attention heads contribute equally to the visual interpretation. However, our study reveals that different heads may capture distinct visual semantics and inherently play distinct roles in visual processing. In light of this observation, we propose HAWK, a head importance-aware visual token pruning method that perceives the varying importance of attention heads in visual tasks to maximize the retention of crucial tokens. By leveraging head importance weights and text-guided attention to assess visual token significance, HAWK effectively retains task-relevant visual tokens while removing redundant ones. The proposed HAWK is entirely training-free and can be seamlessly applied to various MLLMs. Extensive experiments on multiple mainstream vision-language benchmarks demonstrate that HAWK achieves state-of-the-art accuracy. When applied to Qwen2.5-VL, HAWK retains 96.0% of the original accuracy after pruning 80.2% of the visual tokens. Additionally, it reduces end-to-end latency to 74.4% of the original and further decreases GPU memory usage across the tested models. The code is available at https://github.com/peppery77/HAWK.git.

#### 1. 核心问题与整体含义（研究动机和背景）
*   **研究背景**：多模态大语言模型（MLLM）在处理高分辨率图像或长视频时，会产生大量的视觉 Token（通常远超文本 Token）。由于 Transformer 的注意力机制具有平方级计算复杂度，这些冗余的视觉 Token 导致了巨大的推理延迟和显存开销。
*   **核心问题**：现有的视觉 Token 剪枝方法（如基于相似度或简单注意力平均的方法）通常假设所有注意力头对视觉理解的贡献是均等的。
*   **研究动机**：作者通过实验发现，MLLM 中的不同注意力头在处理视觉语义时扮演着截然不同的角色（有的头至关重要，有的则几乎无贡献）。因此，忽略注意力头的重要性差异会导致误删关键信息或保留冗余信息。

#### 2. 方法论：核心思想与关键技术
HAWK 提出了一种**无需训练（Training-free）**的剪枝方案，其核心流程如下：
*   **静态注意力头权重计算（Static Head Importance）**：
    *   通过离线分析，系统性地逐一屏蔽（Ablation）模型中的注意力头，观察其在基准数据集上的性能下降程度。
    *   根据性能下降幅度为每个头分配权重 $w_i$，量化其对视觉任务的贡献。
*   **动态文本引导注意力（Dynamic Text-Guided Attention）**：
    *   利用 LLM 第一层的 Query（来自文本指令）和 Key（来自视觉 Token）计算注意力分数。
    *   **关键细节**：在计算时主动移除旋转位置编码（RoPE），以消除位置偏置，确保注意力分数纯粹反映文本与视觉语义的相关性。
*   **重要性感知剪枝（Head Importance-aware Pruning）**：
    *   将动态注意力分数与静态头权重进行加权求和，得到每个视觉 Token 的最终重要性得分。
    *   根据预设的剪枝比例，保留得分最高的 Top-K 个 Token，其余丢弃。

*   **性能领先**：在相同剪枝率下，HAWK 的准确率一致优于现有 SOTA 方法。例如，在 Qwen2.5-VL 上剪枝 80.2% 的 Token 后，仍能保持 96.0% 的原始精度。

#### 7. 优点
*   **高效且无损**：在极高剪枝率下依然能维持极高的模型性能。
*   **通用性与易用性**：完全无需训练或微调，可直接应用于各种基于 Transformer 架构的 MLLM。

#### 8. 不足与局限
*   **离线依赖**：虽然开销很小，但仍需要针对特定模型进行一次性的离线权重标定。

*   **层级限制**：目前主要利用第一层注意力进行决策，尚未探讨多层融合或深层特征对剪枝决策的潜在增益。

## 摘要
自动特征生成无需人工干预即可从原始表格数据中提取信息丰富的特征，对于实现准确且泛化能力强的机器学习至关重要。传统方法依赖于预定义的算子库，且无法利用任务语义，限制了其为复杂任务生成多样化、高价值特征的能力。近期基于大语言模型（LLM）的方法引入了更丰富的语义信号，但由于固定的生成模式以及缺乏来自学习目标的反馈，仍面临特征空间受限的问题。为了应对这些挑战，我们提出了一种用于自动特征生成的记忆增强型大语言模型多智能体系统（MALMAS）。MALMAS 将生成过程分解为具有不同职责的智能体，并通过路由智能体（Router Agent）在每次迭代中激活合适的智能体子集，从而进一步拓宽了对特征空间的探索。我们进一步集成了一个包含程序性记忆、反馈记忆和概念性记忆的记忆模块，实现了迭代优化，能够自适应地指导后续的特征生成，并提高特征的质量和多样性。在多个公开数据集上与最先进的基准方法进行的广泛实验证明了我们方法的有效性。代码可在 https://github.com/fxdong24/MALMAS 获取。

## Abstract
Automated feature generation extracts informative features from raw tabular data without manual intervention and is crucial for accurate, generalizable machine learning. Traditional methods rely on predefined operator libraries and cannot leverage task semantics, limiting their ability to produce diverse, high-value features for complex tasks. Recent Large Language Model (LLM)-based approaches introduce richer semantic signals, but still suffer from a restricted feature space due to fixed generation patterns and from the absence of feedback from the learning objective. To address these challenges, we propose a Memory-Augmented LLM-based Multi-Agent System (\textbf{MALMAS}) for automated feature generation. MALMAS decomposes the generation process into agents with distinct responsibilities, and a Router Agent activates an appropriate subset of agents per iteration, further broadening exploration of the feature space. We further integrate a memory module comprising procedural memory, feedback memory, and conceptual memory, enabling iterative refinement that adaptively guides subsequent feature generation and improves feature quality and diversity. Extensive experiments on multiple public datasets against state-of-the-art baselines demonstrate the effectiveness of our approach. The code is available at https://github.com/fxdong24/MALMAS

### 论文总结：基于记忆增强大模型多智能体系统的表格数据自动特征生成 (MALMAS)

#### 1. 核心问题与研究动机
*   **核心问题**：如何在表格数据建模中实现高质量、具备语义感知且能自适应优化的自动特征生成（AFG）。
*   **研究动机**：
    *   **传统方法局限性**：如 DFS、OpenFE 等方法依赖预定义的数学算子，缺乏对数据实际业务语义的理解，导致搜索空间受限。
    *   **现有 LLM 方法不足**：虽然引入了语义，但通常采用单智能体或固定模式，缺乏来自下游模型性能的反馈闭环，导致生成过程盲目且容易陷入局部最优。

#### 2. 方法论
MALMAS 框架通过多智能体协作和多级记忆机制来解决上述问题：
*   **核心思想**：将特征生成任务分解给多个专业化的智能体，并通过记忆模块记录“尝试过什么”、“什么有效”以及“为什么有效”，实现迭代式的策略改进。
*   **关键技术细节**：
    *   **多智能体池（Agent Pool）**：包含一元特征、交叉组合、时序特征、聚合构造、局部转换和局部模式等 6 类专业智能体。
    *   **路由智能体（Router Agent）**：根据任务元数据和历史记忆，动态选择当前轮次最合适的智能体子集，提高探索效率。
    *   **三层记忆架构**：
        1.  **程序性记忆（Procedural Memory）**：记录已执行的转换操作，防止重复无效探索。
        2.  **反馈记忆（Feedback Memory）**：记录特征在下游模型中的性能表现（如 AUC 增益），实现信用分配。
        3.  **概念性记忆（Conceptual Memory）**：由总结智能体（Summary Agent）从历史中抽象出高层启发式规则，指导后续生成。
*   **算法流程**：路由选择 -> 智能体并行生成特征 -> 下游模型评估 -> 更新局部与全局记忆 -> 进入下一轮迭代（通常为 4 轮）。

#### 3. 实验设计
*   **数据集**：涵盖 16 个分类数据集和 7 个回归数据集（源自 Kaggle、UCI、OpenML），包括金融、医疗、电商等多个领域。
*   **基准方法（Benchmarks）**：
    *   **传统方法**：DFS, AutoFeat, OpenFE。
    *   **基于 LLM 的方法**：CAAFE, OCTree, LLMFE。
*   **对比维度**：预测性能（AUC、ACC、NRMSE）、消融实验、参数敏感性、跨模型泛化性、以及与 H2O 等经典 AutoML 系统的集成效果。

#### 4. 资源与算力
*   **硬件环境**：Intel Xeon Gold 6326 CPU @ 2.90GHz，**NVIDIA RTX A6000 GPU**。
*   **模型后端**：主要使用 **DeepSeek-V3** API，部分实验对比了 GPT-4o-mini。
*   **开销统计**：平均每个数据集生成特征耗时约 **0.452 小时**，消耗约 **14.76 万个 Token**，单任务成本约 0.17 美元。

#### 5. 实验数量与充分性
*   **实验规模**：共在 23 个数据集上进行了测试，每组实验重复 3 次取平均值。
*   **充分性评价**：
    *   **非常充分**：不仅对比了 SOTA 方法，还进行了详尽的消融实验（逐个添加智能体、逐个移除记忆组件）。
    *   **客观性**：通过 Mean Rank（平均排名）进行全局评价，并验证了生成特征在不同下游模型（XGBoost, LightGBM, MLP 等）上的通用性，证明了实验结果并非过拟合于特定模型。

#### 6. 主要结论与发现
*   **性能领先**：MALMAS 在分类和回归任务上的平均排名均位列第一，显著优于传统和现有的 LLM 方法。
*   **记忆的重要性**：消融实验显示，记忆模块（尤其是全局概念记忆）对性能提升贡献最大，能有效引导 LLM 避开无效搜索。
*   **多智能体协同**：随着激活智能体种类的增加，特征空间的多样性显著提升，模型性能随之增强。
*   **良好集成性**：将 MALMAS 生成的特征输入 H2O AutoML，能进一步提升自动化建模的上限。

#### 7. 优点（亮点）
*   **首创性**：提出了首个用于自动特征生成的记忆增强型多智能体框架。
*   **自适应能力**：通过反馈回路实现了“边做边学”，解决了 LLM 在特征工程中生成逻辑僵化的问题。
*   **实用性强**：计算成本相对较低（基于 DeepSeek），且生成的特征具有一定的可解释性（通过记忆追踪）。

#### 8. 不足与局限
*   **标签依赖**：该方法强依赖于下游评估信号，在极小样本或无标签场景下无法发挥作用。
*   **计算瓶颈**：随着候选特征数量增加，频繁调用下游模型进行验证会产生较大的计算开销。
*   **解释性局限**：虽然有转换描述，但对于极其复杂的非线性组合特征，LLM 仍难以提供深层的数学或物理意义解释。
*   **模态限制**：目前仅针对表格数据，尚未扩展到图像、文本等跨模态特征生成。

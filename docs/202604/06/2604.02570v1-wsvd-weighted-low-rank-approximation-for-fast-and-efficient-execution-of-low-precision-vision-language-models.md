## 摘要
奇异值分解 (SVD) 已成为减轻视觉语言模型 (VLM) 计算负担的重要技术，VLM 在图像描述和视觉问答等任务中起着核心作用。尽管之前的多项研究已提出高效的 SVD 变体以实现低秩操作，但我们发现在实践中，在模型执行期间实现显著的延迟降低仍然具有挑战性。为了解决这一局限性，我们引入了一种新的计算模式，并以更细的粒度应用 SVD，从而在执行延迟方面实现了真实且可衡量的改进。此外，考虑到权重元素在相对重要性上存在差异，我们在 SVD 过程中自适应地为每个元素分配相对重要性，以更好地保持准确性，随后通过对权重和激活同时应用量化来扩展该框架，从而构建出高效的 VLM。总而言之，我们提出了加权 SVD (WSVD)，该方法在保持准确性的同时实现了超过 1.8 倍的解码加速，优于其他方法。我们在 https://github.com/SAI-Lab-NYU/WSVD 开源了代码。

## Abstract
Singular Value Decomposition (SVD) has become an important technique for reducing the computational burden of Vision Language Models (VLMs), which play a central role in tasks such as image captioning and visual question answering. Although multiple prior works have proposed efficient SVD variants to enable low-rank operations, we find that in practice it remains difficult to achieve substantial latency reduction during model execution. To address this limitation, we introduce a new computational pattern and apply SVD at a finer granularity, enabling real and measurable improvements in execution latency. Furthermore, recognizing that weight elements differ in their relative importance, we adaptively allocate relative importance to each element during SVD process to better preserve accuracy, then extend this framework with quantization applied to both weights and activations, resulting in a highly efficient VLM. Collectively, we introduce~\textit{Weighted SVD} (WSVD), which outperforms other approaches by achieving over $1.8\times$ decoding speedup while preserving accuracy. We open source our code at: \href{https://github.com/SAI-Lab-NYU/WSVD}{\texttt{https://github.com/SAI-Lab-NYU/WSVD}

### 1. 论文的核心问题与整体含义（研究动机和背景）
*   **核心问题**：虽然奇异值分解（SVD）被广泛用于视觉语言模型（VLM）的低秩压缩，但在实际部署中，传统的 SVD 方案往往无法显著降低延迟，甚至可能因 KV 缓存重建过程中的频繁内存访问而增加计算开销。
*   **研究动机**：VLM 在处理长图像序列时面临严重的内存带宽瓶颈。作者发现，现有的低秩分解方法在解码阶段需要重新计算 KV 向量，导致内存流量增加。因此，需要一种既能保持精度，又能真正实现硬件加速的压缩方案。

### 2. 论文提出的方法论：WSVD
WSVD 框架包含三个核心技术组件：
*   **细粒度逐头 SVD（Per-head SVD）**：
    *   不同于传统对整个 QKV 矩阵进行 SVD，WSVD 针对每个注意力头（Attention Head）独立进行分解。
    *   **优势**：消除了加载大型共享潜在表示（Latent Representation）的需要，每个头只需加载极小的头相关潜在向量，显著减少了 I/O 流量和重建计算量。
*   **局部加权微调（Weighted Local Finetuning）**：
    *   利用 **Fisher 信息矩阵（FIM）** 评估权重元素的重要性。通过计算校准数据集上的梯度平方期望，识别出对模型精度影响较大的“超级权重”。
    *   在 SVD 分解后，通过最小化加权误差函数 $\min \| F^{1/2} \odot (W - AB) \|^2$ 来微调低秩矩阵 $A$ 和 $B$，以补偿截断误差。
*   **量化感知微调（QAT）与离群值平滑**：
    *   引入正交旋转矩阵（如 Hadamard 矩阵）来平滑激活值中的离群点。
    *   在低秩分解的基础上应用 W8A8 或 W8A4 量化，并通过局部 QAT 进一步恢复精度。
*   **系统级融合算子（Fused Kernel）**：
    *   使用 Triton 开发了定制化内核，将低秩重建过程直接集成到 **Flash Decoding** 流水中。重建后的 KV 块直接留在片上缓存（On-chip Buffer）中参与注意力计算，避免了写回显存的开销。

*   **校准集依赖**：虽然使用了较小的校准集，但模型性能仍可能受到校准数据分布的影响。

## 摘要
递归或循环语言模型最近作为一种新的扩展轴出现，通过在潜在状态上迭代优化相同的模型计算来深化推理。我们将这种扩展原则从单一模型扩展到多智能体系统，并提出疑问：智能体协作本身是否可以通过递归进行扩展？为此，我们引入了 RecursiveMAS，这是一个递归多智能体框架，将整个系统建模为统一的潜在空间递归计算。RecursiveMAS 通过轻量级的 RecursiveLink 模块将异构智能体连接为协作循环，实现了分布内潜在思维生成和跨智能体潜在状态传输。为了优化我们的框架，我们开发了一种内外循环学习算法，通过跨递归轮次的共享基于梯度的信用分配，实现迭代的全系统协同优化。运行复杂度和学习动力学的理论分析表明，RecursiveMAS 比标准的基于文本的多智能体系统（MAS）更高效，并在递归训练期间保持稳定的梯度。在实验方面，我们在 4 种代表性的智能体协作模式下实例化了 RecursiveMAS，并在涵盖数学、科学、医学、搜索和代码生成的 9 个基准测试中进行了评估。与先进的单/多智能体及递归计算基准相比，RecursiveMAS 一致地实现了 8.3% 的平均准确率提升，同时端到端推理速度提高了 1.2 倍至 2.4 倍，Token 使用量减少了 34.6% 至 75.6%。代码和数据可在 https://recursivemas.github.io 获取。

## Abstract
Recursive or looped language models have recently emerged as a new scaling axis by iteratively refining the same model computation over latent states to deepen reasoning. We extend such scaling principle from a single model to multi-agent systems, and ask: Can agent collaboration itself be scaled through recursion? To this end, we introduce RecursiveMAS, a recursive multi-agent framework that casts the entire system as a unified latent-space recursive computation. RecursiveMAS connects heterogeneous agents as a collaboration loop through the lightweight RecursiveLink module, enabling in-distribution latent thoughts generation and cross-agent latent state transfer. To optimize our framework, we develop an inner-outer loop learning algorithm for iterative whole-system co-optimization through shared gradient-based credit assignment across recursion rounds. Theoretical analyses of runtime complexity and learning dynamics establish that RecursiveMAS is more efficient than standard text-based MAS and maintains stable gradients during recursive training. Empirically, we instantiate RecursiveMAS under 4 representative agent collaboration patterns and evaluate across 9 benchmarks spanning mathematics, science, medicine, search, and code generation. In comparison with advanced single/multi-agent and recursive computation baselines, RecursiveMAS consistently delivers an average accuracy improvement of 8.3%, together with 1.2$\times$-2.4$\times$ end-to-end inference speedup, and 34.6%-75.6% token usage reduction. Code and Data are provided in https://recursivemas.github.io.

#### 1. 核心问题与整体含义
论文的研究动机源于单体大语言模型（LLM）在处理复杂任务时存在的推理深度不足、生成近视及搜索效率低等局限。虽然多智能体系统（MAS）通过协作扩展了性能，但传统的基于文本的协作模式存在两大痛点：
*   **效率低下**：智能体间频繁的文本解码与重新编码引入了巨大的推理延迟和 Token 消耗。
*   **优化困难**：文本中介导致梯度断裂，难以对整个系统进行端到端的协同优化。
论文提出 **RecursiveMAS**，旨在将递归计算（Recursive Computation）从单模型扩展到多智能体系统，通过在**隐空间（Latent Space）**进行递归协作，实现性能与效率的双重扩展。

#### 2. 方法论
RecursiveMAS 将整个多智能体系统视为一个统一的隐空间递归计算过程。
*   **核心组件：RecursiveLink**
    *   **Inner Link（内环链接）**：在单个智能体内，将最后一层的隐藏状态映射回输入嵌入空间，支持“潜在思维（Latent Thoughts）”的自回归生成，无需显式解码。
    *   **Outer Link（外环链接）**：连接异构智能体（不同架构、不同维度），实现跨智能体的隐状态传输。采用两层残差投影结构，保持语义稳定。
*   **协作流程**：智能体按预设模式（如顺序、混合等）组成环路。在中间递归轮次中，所有协作均在隐空间完成；仅在最终轮次的最后一个智能体处进行文本解码。
*   **内外环学习算法（Inner-Outer Loop Training）**：
    *   **内环训练**：并行训练每个智能体的 Inner Link，使其生成的隐状态与目标语义分布对齐。
    *   **外环训练**：在系统层面进行递归优化，通过跨轮次的梯度回传（Backpropagation through recursion rounds）对 Outer Link 进行协同微调。
*   **理论支撑**：论文证明了隐空间递归相比文本递归具有更低的运行复杂度（$\Theta(d_h^2)$ vs $\Theta(|V|d_h)$），且能保持梯度稳定性，避免文本 SFT 中的梯度消失问题。

*   **协作模式**：实例化了四种模式：顺序式（Sequential）、混合式（Mixture）、蒸馏式（Distillation）和审议式（Deliberation）。
*   **对比方法**：

    *   **MAS 框架**：Mixture-of-Agents (MoA)、TextGrad。

*   **硬件**：实验使用了 **NVIDIA H100 和 A100 GPU**。

论文进行了多维度的充分实验：
*   **跨领域验证**：在 9 个不同性质的数据集上验证了通用性。
*   **消融实验**：探讨了 RecursiveLink 的架构设计（残差连接的必要性）、潜在思维的长度（$m$ 值）对性能的影响。
*   **扩展性分析**：研究了训练递归轮次与推理递归轮次之间的“扩展定律（Scaling Law）”。

#### 6 主要结论与发现
*   **性能提升**：RecursiveMAS 相比最强基准平均准确率提升 **8.3%**。
*   **高效推理**：实现 **1.2x - 2.4x** 的端到端推理加速，并减少了 **34.6% - 75.6%** 的 Token 使用量。
*   **递归扩展性**：随着递归深度增加，性能呈上升趋势，且隐空间协作的效率优势随深度增加而扩大。
*   **语义对齐**：可视化分析显示，随着递归轮次增加，系统生成的隐状态分布逐渐向真实答案（Ground Truth）的语义分布靠拢。

#### 7. 优点
*   **架构无关性**：能够灵活适配顺序、并行、蒸馏等多种多智能体拓扑结构。
*   **极高效率**：通过隐空间通信避开了 LLM 推理中最耗时的词表采样阶段。
*   **学习稳定性**：理论与实验均证明了其在长链递归优化中的梯度优势。
*   **低成本**：仅需微调极少量的投影层参数即可激活强大的协作能力。

*   **潜在思维长度限制**：虽然实验显示 $m=80$ 左右性能趋于稳定，但对于极复杂任务，固定长度的隐状态可能仍存在信息瓶颈。

*   **黑盒特性**：隐空间协作虽然高效，但相比文本协作失去了中间步骤的可解释性（人类无法直接阅读中间轮次的“潜在思维”）。

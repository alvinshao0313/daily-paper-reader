## 摘要
智能体技能是程序性知识和可执行资源的结构化包，由智能体在推理时动态加载，已成为增强大语言模型（LLM）智能体的一种可靠机制。然而，推理阶段的技能增强存在根本局限：检索噪声会引入无关引导，注入的技能内容会产生巨大的 Token 开销，且模型从未真正习得其所遵循的知识。我们探究技能是否可以内化到模型参数中，从而在无需任何运行时技能检索的情况下实现零样本自主行为。我们提出了 SKILL0，这是一个专为技能内化设计的上下文内强化学习框架。SKILL0 引入了一种训练期课程，从完整的技能上下文开始并逐步撤除。技能在离线状态下按类别分组，并与交互历史共同渲染为紧凑的视觉上下文，以教导模型工具调用和多轮任务完成。随后，动态课程会评估每个技能文件的同策略（on-policy）辅助作用，在随线性衰减的预算内，仅保留当前策略仍能从中获益的技能，直到智能体在完全零样本的设置下运行。广泛的智能体实验表明，SKILL0 相比标准强化学习基线取得了显著提升（ALFWorld 提升 9.7%，Search-QA 提升 6.6%），同时保持了每步少于 0.5k Token 的高效上下文。我们的代码开源于 https://github.com/ZJU-REAL/SkillZero。

## Abstract
Agent skills, structured packages of procedural knowledge and executable resources that agents dynamically load at inference time, have become a reliable mechanism for augmenting LLM agents. Yet inference-time skill augmentation is fundamentally limited: retrieval noise introduces irrelevant guidance, injected skill content imposes substantial token overhead, and the model never truly acquires the knowledge it merely follows. We ask whether skills can instead be internalized into model parameters, enabling zero-shot autonomous behavior without any runtime skill retrieval. We introduce SKILL0, an in-context reinforcement learning framework designed for skill internalization. SKILL0 introduces a training-time curriculum that begins with full skill context and progressively withdraws it. Skills are grouped offline by category and rendered with interaction history into a compact visual context, teaching he model tool invocation and multi-turn task completion. A Dynamic Curriculum then evaluates each skill file's on-policy helpfulness, retaining only those from which the current policy still benefits within a linearly decaying budget, until the agent operates in a fully zero-shot setting. Extensive agentic experiments demonstrate that SKILL0 achieves substantial improvements over the standard RL baseline (+9.7\% for ALFWorld and +6.6\% for Search-QA), while maintaining a highly efficient context of fewer than 0.5k tokens per step. Our code is available at https://github.com/ZJU-REAL/SkillZero.

### 1. 核心问题与研究动机
*   **核心问题**：当前的 LLM 智能体主要依赖“推理时技能增强”（Inference-time Skill Augmentation），即通过检索技能库并将其注入 Prompt 来引导智能体。这种方式存在三大局限：
    1.  **检索噪声**：引入无关或误导性信息。
    2.  **Token 开销**：多轮对话中注入技能描述会导致上下文急剧膨胀。
    3.  **缺乏真正习得**：模型只是在“遵循”指令而非“掌握”技能，知识存在于上下文而非参数中。
*   **研究动机**：探索能否通过强化学习（RL）将外部技能“内化”到模型参数中，使智能体在推理阶段无需检索即可实现零样本（Zero-shot）的自主行为。

### 2. 方法论
SKILL0 提出了**上下文内智能体强化学习（In-Context Agentic RL）**框架，核心包含以下三个关键技术：
*   **上下文内强化学习 (ICRL)**：在训练的 Rollout 阶段提供结构化的技能引导，但在推理阶段完全移除。通过 RL 优化目标，驱动模型从“依赖上下文执行”转向“内在能力驱动”。
*   **视觉上下文渲染与自压缩**：为了解决多域扩展时的 Token 成本问题，SKILL0 将文本交互历史和技能描述渲染为紧凑的 RGB 图像，并通过视觉编码器（Vision Encoder）处理。模型可以自发生成压缩率 $c_t$，在保证决策信息完整的同时极大地降低 Token 消耗。
*   **动态课程学习 (Dynamic Curriculum)**：
    *   **离线技能分组**：按类别将技能文件与特定的验证子任务关联。
    *   **辅助性驱动的退火机制**：引入线性衰减的技能预算 $M$。在训练过程中，通过对比“有技能”和“无技能”下的表现差异来评估每个技能文件的“同策略辅助性”（On-policy Helpfulness）。
    *   **过滤与筛选**：仅保留当前策略仍能获益的技能，直到预算归零，最终实现完全脱离外部技能的推理。

### 5. 实验数量与充分性
*   **实验规模**：涵盖了具身决策和复杂问答两大领域，使用了 3B 和 7B 两种参数规模的模型。
*   **消融实验**：针对技能预算 $M$ 的设置、动态课程策略（过滤、排序、选择）以及验证间隔 $d$ 进行了详细的消融研究。
*   **充分性评价**：实验设计较为全面，不仅对比了最终性能，还详细展示了训练过程中的奖励曲线和技能“辅助性”的变化趋势。通过与闭源模型（GPT-4o, Gemini-2.5-Pro）及多种记忆增强/搜索增强方法的对比，证明了其竞争力和客观性。

### 6. 主要结论与发现
*   **性能提升**：SKILL0 在推理时不使用任何技能检索的情况下，表现优于标准 RL 基线（ALFWorld +9.7%，Search-QA +6.6%）。
*   **技能内化趋势**：实验观察到明显的“先升后降”模式——早期模型学习利用技能 Prompt，后期随着课程推进，模型在无 Prompt 下的表现逐渐追平甚至超越有 Prompt 的表现，证明技能已转化为参数知识。
*   **极高效率**：由于采用了视觉压缩和技能内化，SKILL0 的推理 Token 成本极低（每步 < 0.5k），比 SkillRL 等方法降低了 5 倍以上。

### 7. 优点与亮点
*   **范式创新**：首次将“技能内化”作为显式的 RL 训练目标，解决了智能体对外部 Prompt 的长期依赖问题。
*   **动态自适应**：动态课程学习机制避免了硬编码的撤除计划，能够根据模型当前的实际掌握情况灵活调整训练难度。
*   **视觉压缩方案**：巧妙地利用视觉编码器处理长上下文，在降低成本的同时保留了结构化信息。

### 8. 不足与局限
*   **依赖初始技能库**：SKILL0 的效果高度依赖于初始 SkillBank 的质量，如果初始技能描述有误或不全，内化效果会受限。
*   **离线分组开销**：在应用到新领域时，需要预先进行“相关性驱动的技能分组”，这增加了一定的前期人工或计算成本。
*   **泛化边界**：虽然在多跳 QA 上表现出泛化性，但对于完全未见过的全新技能类型，其零样本表现仍有待进一步验证。

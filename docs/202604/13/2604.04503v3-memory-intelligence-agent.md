## 摘要
深度研究智能体 (DRA) 将大语言模型 (LLM) 推理与外部工具相结合。记忆系统使 DRA 能够利用历史经验，这对于高效推理和自主进化至关重要。现有方法依赖于从记忆中检索相似轨迹来辅助推理，但面临记忆进化效率低下以及存储和检索成本增加的关键局限。为解决这些问题，我们提出了一种新型的记忆智能体 (MIA) 框架，由“管理者-规划者-执行者” (Manager-Planner-Executor) 架构组成。记忆管理者是一个非参数化记忆系统，可存储压缩的历史搜索轨迹；规划者是一个参数化记忆智能体，可为问题生成搜索计划；执行者是另一个在搜索计划指导下搜索和分析信息的智能体。为了构建 MIA 框架，我们首先采用交替强化学习范式来增强规划者与执行者之间的协作。此外，我们使规划者能够在测试时学习过程中持续进化，在不中断推理过程的情况下随推理同步进行实时更新。此外，我们在参数化记忆和非参数化记忆之间建立了双向转换循环，以实现高效的记忆进化。最后，我们引入了反思和无监督判断机制，以增强在开放世界中的推理和自我进化能力。在 11 个基准测试上的广泛实验证明了 MIA 的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

### 1. 核心问题与整体含义（研究动机和背景）
*   **研究背景**：深度研究智能体通过结合大语言模型（LLM）的推理能力与外部工具（如搜索引擎）来处理复杂任务。记忆系统是其积累经验、优化策略的关键。
*   **核心问题**：现有的智能体记忆系统（如长上下文记忆）存在四大局限：
    1.  **注意力稀释**：长上下文干扰模型对当前问题的理解。
    2.  **噪声干扰**：无关或弱相关的历史信息降低推理质量。
    3.  **存储瓶颈**：持续增长的对话历史带来巨大的存储压力。
    4.  **检索低效**：在大规模记忆库中检索耗时且昂贵。
*   **研究动机**：深度研究更依赖“过程导向”的记忆（如何获得结果），而非简单的“事实导向”记忆。因此，需要一种能将历史经验转化为规划指导并实现自我进化的机制。

### 2. 方法论：核心思想与关键技术
MIA 采用了**管理者-规划者-执行者（Manager-Planner-Executor）**的解耦架构：
*   **记忆管理者 (Memory Manager)**：非参数化记忆系统。负责将冗长的搜索轨迹压缩为结构化的“工作流（Workflow）”摘要，存储成功（正向）和失败（负向）的案例。
*   **规划者 (Planner)**：参数化记忆智能体。基于检索到的历史工作流生成针对当前问题的搜索计划，并具备反思（Reflection）与重新规划（Replan）的能力。
*   **执行者 (Executor)**：操作终端。负责解析规划者的指令，通过 ReAct 循环调用工具并进行推理。
*   **关键技术细节**：
    *   **交替强化学习 (Alternating RL)**：基于 GRPO 算法，第一阶段训练执行者遵循规划，第二阶段训练规划者吸收记忆并优化规划逻辑。
    *   **测试时学习 (Test-Time Learning, TTL)**：在推理过程中实时更新规划者的参数，实现“边推理边进化”，无需中断工作流。
    *   **双向记忆转换**：非参数记忆（工作流）为规划提供上下文，而规划者的参数更新则将这些经验内化为模型权重（参数化记忆）。
    *   **无监督判断机制**：模拟学术评审过程（Reviewer-AC 架构），通过逻辑、事实、有效性三个维度的交叉评审，在无标注环境下实现自我进化。

### 6. 主要结论与发现
*   **性能飞跃**：MIA 显著提升了现有最强模型的表现，如将 GPT-5.4 在 LiveVQA 上的准确率提升了 9%。
*   **以小博大**：搭载 MIA 的 **7B 规模执行者** 在平均性能上超过了 **32B 规模** 的同系列模型，性能提升达 31%。
*   **自我进化有效性**：在无监督设置下，MIA 通过多次迭代展现了持续的性能增长（如在 HotpotQA 上从 59.6 提升至 63.2）。
*   **记忆效率**：相比长上下文方法，MIA 通过压缩工作流和参数化内化，有效解决了记忆噪声和存储成本问题。

### 7. 优点（亮点）
*   **架构创新**：解耦了认知（规划）与操作（执行），并引入了类似人类大脑的参数化/非参数化双重记忆机制。
*   **实时进化**：TTL 机制允许智能体在测试阶段根据环境反馈即时优化，增强了对新领域的适应力。
*   **无监督能力**：提出的 Reviewer-AC 评审框架为智能体在缺乏 Ground-truth 的真实世界中自我进化提供了可行路径。

### 8. 不足与局限
*   **推理延迟**：由于引入了规划、执行、反思、再执行的循环，且 TTL 涉及参数更新，单次推理的时间成本可能高于传统 Agent。
*   **模型依赖**：框架的效果高度依赖于作为“管理者”和“评判者”的大规模模型（如 Qwen3-32B）的判断准确性。
*   **工具范围**：目前主要集中在搜索工具，对于更复杂的 API 调用或多工具协同的覆盖尚待进一步验证。

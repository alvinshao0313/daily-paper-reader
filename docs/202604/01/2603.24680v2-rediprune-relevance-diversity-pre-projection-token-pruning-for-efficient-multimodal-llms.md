## Abstract
Recent multimodal large language models are computationally expensive because Transformers must process a large number of visual tokens. We present ReDiPrune, a training-free token pruning method applied before the vision-language projector, where visual features remain rich and discriminative. Unlike post-projection pruning methods that operate on compressed representations, ReDiPrune selects informative tokens directly from vision encoder outputs, preserving fine-grained spatial and semantic cues. Each token is scored by a lightweight rule that jointly consider text-conditioned relevance and max-min diversity, ensuring the selected tokens are both query-relevant and non-redundant. ReDiPrune is fully plug-and-play, requiring no retraining or architectural modifications, and can be seamlessly inserted between the encoder and projector. Across four video and five image benchmarks, it consistently improves the accuracy-efficiency trade-off. For example, on EgoSchema with LLaVA-NeXT-Video-7B, retaining only 15% of visual tokens yields a +2.0% absolute accuracy gain while reducing computation by more than $6\times$ in TFLOPs. Code is available at https://github.com/UA-CVML/ReDiPrune.

### 1. 核心问题与研究背景
*   **核心问题**：多模态大语言模型（MLLM）在处理高分辨率图像或长视频时，会产生海量的视觉Token，导致Transformer注意力的计算开销呈平方级增长，推理速度慢且显存占用高。
*   **研究背景**：现有的Token剪枝方法大多在“投影后”（Post-projection）进行，此时视觉特征已与语言空间混合或被压缩，容易丢失细粒度的空间和语义细节。而现有的“投影前”方法往往需要昂贵的重新训练或微调。

### 2. 方法论
*   **核心思想**：提出一种**无需训练（Training-free）**、**即插即用**的预投影剪枝模块。在视觉编码器输出后、投影器（Projector）输入前，直接筛选出与文本查询相关且具有多样性的视觉Token。
*   **关键技术细节**：
    *   **提示词驱动的查询构建**：通过对文本提示词进行加权聚合（如指数加权），生成一个标准化的查询向量 $\hat{q}$。
    *   **文本相关性评分（Relevance）**：计算视觉Token与查询向量 $\hat{q}$ 之间的余弦相似度。
    *   **最大最小多样性评分（Diversity）**：计算视觉Token之间的余弦不相似度，避免选择冗余的重复区域。
    *   **统一目标函数与贪婪求解**：定义一个联合目标函数，平衡相关性和多样性。采用贪婪算法逐一选择Token：首先选出最相关的Token，随后迭代选择能最大化“与已选集合的最小距离 + 相关性权重”的Token。
    *   **候选预过滤**：引入阈值 $\tau$ 过滤掉低相关性Token，提高处理效率。

*   **数据集与场景**：

### 4. 资源与算力
*   **硬件环境**：实验在 **4张 NVIDIA A100 80GB GPU** 上进行。
*   **训练时长**：由于该方法是**无需训练（Training-free）**的，因此不存在训练耗时。

### 5. 实验数量与充分性
*   **实验规模**：涵盖了9个主流多模态基准测试，涵盖了视频理解、图像问答、物体幻觉评估等多个维度。
*   **消融实验**：针对不同的查询加权策略（如平均加权、自注意力加权、指数加权等）以及剪枝对注意力分布的影响进行了详细分析。

### 6. 主要结论与发现
*   **精度不降反升**：在大幅减少Token的情况下，模型性能往往有所提升。例如在EgoSchema上，仅保留15%的Token，准确率反而提升了2.0%。这表明剪枝起到了“去噪”作用，帮助模型聚焦关键信息。
*   **高效性**：在视频任务中，ReDiPrune能实现超过6倍的计算量（TFLOPs）削减和显著的推理加速。
*   **预投影的优势**：相比投影后剪枝，在原始视觉特征空间进行筛选能更好地保留细粒度语义，尤其在处理需要精确识别（如识别视频中的微小物体）的任务时表现更优。

### 7. 优点
*   **无需训练**：无需对现有模型进行任何微调，极大降低了部署成本。
*   **即插即用**：可以无缝集成到任何类LLaVA架构的模型中。
*   **语义保留好**：通过在投影前操作，避免了模态混合导致的视觉特征模糊。
*   **综合性评分**：同时考虑了查询相关性和视觉多样性，比单一维度的剪枝更鲁棒。

### 8. 不足与局限
*   **架构局限性**：目前主要针对类LLaVA架构（视觉编码器+线性投影器+LLM）进行验证，对于更复杂的交叉注意力架构（如Flamingo）的适用性有待研究。
*   **帧内剪枝**：目前的策略主要基于单帧内的Token筛选，尚未实现全局的时空联合剪枝（Spatio-temporal pruning）。
*   **固定预算**：剪枝比例（Keep Ratio）需要预先设定，缺乏根据输入内容复杂度动态调整Token数量的自适应机制。

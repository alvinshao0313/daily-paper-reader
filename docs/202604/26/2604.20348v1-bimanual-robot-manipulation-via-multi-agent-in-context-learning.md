## 摘要
大语言模型（LLMs）已成为具身控制的强大推理引擎。特别是，上下文学习（ICL）使现成的纯文本 LLM 能够在无需任何特定任务训练的情况下预测机器人动作，同时保留其泛化能力。将 ICL 应用于双臂操作仍具挑战性，因为高维联合动作空间和紧密的臂间协调约束会迅速超出标准上下文窗口的负荷。为此，我们引入了 BiCICLe（双臂协调上下文学习），这是首个使标准 LLM 能够在无需微调的情况下执行少样本双臂操作的框架。BiCICLe 将双臂控制建模为多智能体主从（leader-follower）问题，将动作空间解耦为序列化的、有条件的单臂预测。这自然地扩展到了“双臂辩论”（Arms' Debate）——一种迭代优化过程，并引入了第三个 LLM 作为裁判（LLM-as-Judge）来评估并选择最合理的协调轨迹。在 TWIN 基准测试的 13 个任务中进行的评估显示，BiCICLe 实现了高达 71.1% 的平均成功率，比表现最好的免训练基准方法高出 6.7 个百分点，并超过了大多数监督学习方法。我们进一步展示了在面对新任务时强大的少样本泛化能力。

## Abstract
Language Models (LLMs) have emerged as powerful reasoning engines for embodied control. In particular, In-Context Learning (ICL) enables off-the-shelf, text-only LLMs to predict robot actions without any task-specific training while preserving their generalization capabilities. Applying ICL to bimanual manipulation remains challenging, as the high-dimensional joint action space and tight inter-arm coordination constraints rapidly overwhelm standard context windows. To address this, we introduce BiCICLe (Bimanual Coordinated In-Context Learning), the first framework that enables standard LLMs to perform few-shot bimanual manipulation without fine-tuning. BiCICLe frames bimanual control as a multi-agent leader-follower problem, decoupling the action space into sequential, conditioned single-arm predictions. This naturally extends to Arms' Debate, an iterative refinement process, and to the introduction of a third LLM-as-Judge to evaluate and select the most plausible coordinated trajectories. Evaluated on 13 tasks from the TWIN benchmark, BiCICLe achieves up to 71.1% average success rate, outperforming the best training-free baseline by 6.7 percentage points and surpassing most supervised methods. We further demonstrate strong few-shot generalization on novel tasks.

### 1. 核心问题与整体含义（研究动机和背景）
双臂机器人操作（如拧瓶盖、抬托盘）要求两臂在空间、时间和力量上高度同步。相比单臂控制，双臂操作面临**高维动作空间**（联合空间随维度指数增长）和**严苛的协调约束**。
*   **研究动机**：现有的模仿学习或强化学习方法通常需要海量任务特定数据。大语言模型（LLM）的上下文学习（ICL）虽能实现少样本泛化，但在双臂任务中，将双臂动作简单拼接会导致文本序列过长，超出 LLM 的上下文窗口，且难以捕捉复杂的臂间依赖关系。
*   **核心问题**：如何在不进行模型微调的前提下，利用 LLM 的推理能力实现高效、协调的双臂操作。

### 2. 方法论
论文提出了 **BiCICLe** 框架，其核心思想是将双臂控制解耦为多智能体协作问题：
*   **主从分解（Leader-Follower Decomposition）**：将双臂的 $Z^{14}$ 动作空间分解为两个顺序执行的 $Z^7$ 预测。首先由“领导者”手臂根据场景观察预测其完整轨迹；随后“跟随者”手臂在观察基础上，将领导者的计划作为额外条件进行预测。
*   **双臂辩论（Arms’ Debate）**：一种推理时优化策略。领导者和跟随者进行多轮迭代，每一轮都将对方最新的轨迹作为时空参考进行重新规划，从而增强同步性。
*   **LLM 评判机制（Best-of-N via LLM-as-Judge）**：生成多个候选轨迹对，引入第三个 LLM 智能体作为裁判，根据演示样本的逻辑一致性对候选方案进行评分，选择得分最高的协调计划。
*   **表示方法**：将场景简化为文本。物体位置被离散化为 $100^3$ 的体素网格，旋转角度离散化为 5° 间隔的箱位，夹持器状态为二进制。

### 3. 实验设计
*   **数据集/场景**：主要在 **TWIN 基准测试**（包含 13 个双臂操作任务，如推箱子、移交物品、抬托盘等）中进行仿真评估。
*   **对比方法**：
    *   **免训练基准**：RoboPrompt（单智能体/独立双智能体）、KAT（基于视觉特征点）、VLM-LF（基于视觉语言模型）。

*   **泛化性测试**：设计了两个全新的任务（关罐子、从箱中取物）测试 OOD（分布外）泛化能力。

*   **模型后端**：主要使用 **GPT-5-mini**（API 调用）和开源模型 **Qwen 2.5 7B**。

### 5. 实验数量与充分性
*   **实验规模**：每个任务进行 3 个随机种子 × 100 个回合的评估，数据量较为充分。
*   **消融实验**：非常详尽。涵盖了领导者手臂的选择（左/右）、对话式优化与辩论机制的对比、观察表示（是否包含旋转）、点云提取方法等。
*   **客观性**：通过对比不同规模的后端模型（GPT vs Qwen）验证了架构的普适性，并与多种监督学习方法进行了横向对比，实验设计客观公平。

### 6. 主要结论与发现
*   **性能卓越**：BiCICLe + Best-of-N 达到了 **71.1%** 的平均成功率，比最强的免训练基准高出 6.7%，甚至超越了 PerAct 2 和 $\pi_0$-keypose 等监督学习方法。
*   **解耦的必要性**：实验证明，简单的单智能体（SA）处理高维空间效果差，而完全独立的双智能体（DA）缺乏协调。BiCICLe 的主从结构在两者间取得了最佳平衡。
*   **泛化优势**：在全新任务上，BiCICLe 的成功率（54.5%）远高于仅用 10 个样本微调的 SOTA 模型 3DFA（10.0%），证明了 ICL 在少样本场景下的强大潜力。

### 7. 优点
*   **无需训练**：完全依赖现成的 LLM，极大降低了双臂机器人开发的门槛和数据成本。
*   **架构优雅**：通过主从分解巧妙解决了 LLM 上下文窗口限制与高维动作预测的矛盾。
*   **推理时优化**：引入“辩论”和“评判”机制，有效抑制了 LLM 生成的随机性，提升了动作的鲁棒性。

### 8. 不足与局限
*   **精度限制**：依赖于体素化（$100^3$）和角度离散化，对于极精细的操作（如穿针引线、处理细绳）可能存在精度不足。
*   **推理成本**：多次调用 LLM API 带来了较高的推理延迟和 Token 消耗成本，难以满足实时性要求极高的场景。
*   **感知依赖**：实验依赖于准确的物体分割和点云提取，在复杂光照或遮挡严重的真实环境中，感知噪声可能影响性能。

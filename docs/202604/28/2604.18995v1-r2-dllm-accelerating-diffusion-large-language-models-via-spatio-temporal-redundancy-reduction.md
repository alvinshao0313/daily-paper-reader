## 摘要
扩散大语言模型（dLLMs）通过实现并行 Token 预测，已成为自回归生成的一种极具前景的替代方案。然而，实际的 dLLM 解码仍面临高推理延迟的问题，限制了其部署。在这项工作中，我们观察到这种低效很大程度上源于解码过程中反复出现的冗余，包括由置信度簇和位置歧义引起的空间冗余，以及由重复掩码已稳定的预测结果引起的时间冗余。受这些模式的启发，我们提出了 $R^2$-dLLM，这是一个从推理和训练两个角度减少解码冗余的统一框架。在推理阶段，我们引入了无需训练的解码规则，通过聚合局部置信度和 Token 预测，并确定时间上稳定的 Token，从而避免冗余的解码步骤。我们还进一步提出了一种冗余感知的有监督微调流水线，使模型与高效的解码轨迹对齐，并减少对手动调节阈值的依赖。实验表明，与现有的解码策略相比，$R^2$-dLLM 在不同模型和任务中始终能减少高达 75% 的解码步数，同时保持具有竞争力的生成质量。这些结果验证了解码冗余是 dLLM 的核心瓶颈，显式地减少冗余可以带来显著的实际效率提升。

## Abstract
Diffusion Large Language Models (dLLMs) have emerged as a promising alternative to autoregressive generation by enabling parallel token prediction. However, practical dLLM decoding still suffers from high inference latency, which limits deployment. In this work, we observe that a substantial part of this inefficiency comes from recurring redundancy in the decoding process, including spatial redundancy caused by confidence clusters and positional ambiguity, and temporal redundancy caused by repeatedly remasking predictions that have already stabilized. Motivated by these patterns, we propose $R^2$-dLLM, a unified framework for reducing decoding redundancy from both inference and training perspectives. At inference time, we introduce training-free decoding rules that aggregate local confidence and token predictions, and finalize temporally stable tokens to avoid redundant decoding steps. We further propose a redundancy-aware supervised fine-tuning pipeline that aligns the model with efficient decoding trajectories and reduces reliance on manually tuned thresholds. Experiments demonstrate that $R^2$-dLLM consistently reduces the number of decoding steps by up to 75% compared to existing decoding strategies, while maintaining competitive generation quality across different models and tasks. These results validate that decoding redundancy is a central bottleneck in dLLMs, and that explicitly reducing it yields substantial practical efficiency gains.

### 1. 核心问题与整体含义（研究动机和背景）
扩散大语言模型（dLLMs）通过并行预测 Token 克服了自回归（AR）模型逐个生成的瓶颈，但在实际应用中，其推理延迟依然较高。作者发现，这种低效主要源于解码轨迹中的**时空冗余（Spatio-Temporal Redundancy）**：
*   **空间冗余**：模型在相邻位置产生高置信度预测（置信度簇）或在多个相邻位置预测同一个 Token（位置歧义/Token 簇），导致重复解码。
*   **时间冗余**：许多 Token 在解码早期就已稳定，但现有策略仍会不断对其进行“掩码-再预测”的循环。
论文的核心目标是通过显式减少这些冗余来加速 dLLM 的推理过程。

### 2. 方法论：核心思想与关键技术
作者提出了 **$R^2$-dLLM** 统一框架，从推理规则和模型训练两个维度入手：

*   **推理端：免训练解码策略（Training-free Strategies）**
    *   **空间聚合**：
        *   *置信度簇聚合*：计算局部窗口内的平均置信度，若超过阈值 $\tau_s$，则同时固定该窗口内的所有 Token。
        *   *Token 簇聚合*：若同一 Token 在相邻位置重复出现，选择置信度最高的位置固定，消除位置歧义。
    *   **时间一致性检查**：引入多步一致性判断。如果一个位置的预测 Token 在连续 $m$ 步（实验中 $m=3$）内保持不变，且最后一步置信度达标，则提前固定该 Token，避免后续重掩码。

*   **训练端：冗余感知监督微调（Redundancy-aware SFT）**
    *   **数据筛选**：利用上述推理规则对候选生成轨迹进行打分（$R_{total} = R_s + R_t$），优先选择正确且冗余得分最低的轨迹作为训练数据。
    *   **半自回归微调**：采用分块掩码训练协议，使模型在训练阶段就学习如何以更少的步骤、更高效的路径收敛到目标文本。

*   **显著加速**：相比 Vanilla 解码，$R^2$-dLLM 实现了高达 **7.7倍** 的延迟加速；相比 SOTA 的 Fast-dLLM，NFE 减少了 **21%~40%**。

*   **泛化性强**：在不同架构（LLaDA, Dream）和不同任务（数学, 代码）上均表现出一致的增益。

### 8. 不足与局限
*   **任务局限性**：实验主要集中在逻辑性强、答案相对确定的数学和代码任务上，在开放式对话或文学创作等发散性任务上的表现尚待验证。
*   **阈值依赖**：虽然 SFT 减轻了对规则的依赖，但推理时仍涉及 $\tau_s, \tau_t$ 等超参数，不同模型和任务可能需要微调这些阈值以达到最佳平衡。
*   **训练成本**：虽然推理变快了，但 redundancy-aware SFT 需要先生成多条轨迹并打分筛选，增加了前期的离线数据准备成本。

## Abstract
Long-context reasoning is a critical capability of large language models (LLMs), enabling applications such as long-document understanding, summarization, and code generation. However, efficient autoregressive inference relies on the key-value (KV) cache, whose memory footprint grows linearly with sequence length, leading to a major memory bottleneck. To mitigate this overhead, KV cache pruning methods discard cached tokens with low attention scores during inference. Most existing methods apply a uniform pruning ratio across layers, implicitly assuming that all layers contribute equally to overall model performance. We show that this assumption is suboptimal, as layers differ significantly in their sensitivity to pruning. We propose DepthKV, a layer-dependent pruning framework that allocates a fixed global KV budget across layers based on their sensitivity, rather than using a uniform allocation. Across multiple models and tasks, DepthKV consistently outperforms uniform pruning at the same global pruning ratio, demonstrating more effective utilization of the KV cache budget through layer-dependent allocation.

### 1. 论文的核心问题与整体含义
随着大语言模型（LLM）支持的上下文窗口不断扩大，推理过程中的 **KV Cache（键值缓存）** 占用的显存呈线性增长，成为长文本推理的主要瓶颈。现有的 KV Cache 剪枝方法（如 H2O、SnapKV 等）通常假设所有 Transformer 层同等重要，并在所有层应用**统一的剪枝比例**。
本文的研究动机在于挑战这一假设。作者通过实验证明，不同层对剪枝的敏感度存在显著差异，某些特定层（称为“内容放大层”）对维持生成内容的质量和长度至关重要。因此，论文提出了 **DepthKV** 框架，旨在通过**层依赖（Layer-Dependent）**的资源分配，在固定全局显存预算下优化推理性能。

### 2. 论文提出的方法论
DepthKV 的核心思想是：**放弃均匀分配，根据层的敏感度动态调整每一层的 KV 缓存预算。**

*   **重要性评估指标：** 引入了基于表征性质的指标，特别是 **InfoNCE**（对比损失指标）。研究发现，InfoNCE 与层的重要性呈负相关——表征越稳健（InfoNCE 值越低）的层，在剪枝时性能下降越严重，因此需要保留更多缓存。
*   **三种分配策略：**
    1.  **中间层保护 (MLP)：** 鉴于中间层通常承载更多语义信息，直接保留中间几层不剪枝，其余层均匀剪枝。
    2.  **指标引导分配 (MGA)：** 利用 InfoNCE 指标为每层计算重要性得分，按比例分配剪枝率（敏感层保留更多，不敏感层剪枝更多）。
    3.  **混合策略 (MLMA)：** 结合上述两者，保护中间层的同时，对剩余层按指标分配预算。
*   **内容放大层识别：** 论文发现剪枝某些层会导致输出变短且信息量骤减（通过 YapScore 衡量），这些层被识别为关键层，在分配预算时需优先考虑。

*   **实验规模：** 涵盖了 3 个主流模型家族、7 个不同领域的任务数据集。
*   **消融与统计：** 进行了层级敏感度消融研究（每次只剪枝一层），并使用了 **置换检验 (Permutation Test)** 来验证层间差异的统计显著性（p-value < 0.05）。
*   **客观性：** 实验在相同的全局 KV 预算（如 60% 缩减率）下进行对比，确保了公平性。引入 LLM-as-a-judge 进一步增强了自动指标之外的客观评价。

### 6. 主要结论与发现
*   **层间差异显著：** Transformer 各层对 KV 剪枝的敏感度极不均匀，统一剪枝确实是次优的。
*   **内容放大效应：** 关键层的剪枝不仅降低准确性，还会抑制模型生成详细内容的能力。
*   **DepthKV 优越性：** 在所有测试模型和任务中，DepthKV（尤其是 MGA 和 MLMA 变体）在相同内存预算下一致优于均匀剪枝。
*   **指标有效性：** InfoNCE 等表征指标可以作为无需标签的“层重要性”预测器。

### 7. 优点
*   **即插即用：** 作为一种后训练方法，无需重新训练模型，易于集成到现有的推理框架中。
*   **资源利用率高：** 通过精细化分配预算，在不增加额外显存的前提下提升了长文本处理质量。
*   **理论支撑：** 结合了表征学习中的几何与信息论指标，为“为什么某些层更重要”提供了量化解释。

### 8. 不足与局限
*   **非查询感知 (Non-query-aware)：** 剪枝决策是在预填充阶段一次性做出的，没有根据解码过程中的具体查询动态调整，可能漏掉某些在特定解码步骤才变得重要的标记。
*   **缺乏头级粒度：** 目前只在“层”级别分配预算，未考虑同一层内不同注意力头（Attention Heads）的功能差异。
*   **任务依赖性：** 虽然提供了指标引导，但不同任务（如摘要 vs 数学推理）的最优层分配策略仍存在一定波动。

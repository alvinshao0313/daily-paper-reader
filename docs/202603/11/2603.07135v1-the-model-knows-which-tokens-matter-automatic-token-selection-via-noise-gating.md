## 摘要
在视觉语言模型（VLM）中，视觉 token 占据了主要的推理成本，但其中许多包含冗余信息。现有的剪枝方法缓解了这一问题，但通常依赖于注意力权重大小或相似度得分。我们将视觉 token 剪枝重新定义为容量受限的通信问题：在给定固定预算 K 的情况下，模型必须分配有限的带宽以最大限度地保留视觉信息。我们提出了 AutoSelect，它在冻结的 VLM 上附加了一个轻量级的评分器（Scorer）和去噪器（Denoiser），且仅使用标准的下一个 token 预测损失进行训练，无需辅助目标或额外标注。在训练期间，一个方差保持噪声门控根据预测的重要性调节每个 token 的信息流，从而使梯度能够传播到所有 token；随后，一个对角注意力去噪器恢复受扰动的表示。在推理时，仅保留评分器和硬 top-K 选择，增加的延迟微乎其微。在十个 VLM 基准测试中，AutoSelect 在仅增加 0.69 毫秒开销的情况下，保留了完整模型 96.5% 的准确率，同时将 LLM 预填充速度提升了 2.85 倍，并且可以迁移到不同的 VLM 骨干网络，无需针对特定架构进行调整。代码可在 https://github.com/MedHK23/AutoSelect 获取。

## Abstract
Visual tokens dominate inference cost in vision-language models (VLMs), yet many carry redundant information. Existing pruning methods alleviate this but typically rely on attention magnitude or similarity scores. We reformulate visual token pruning as capacity constrained communication: given a fixed budget K, the model must allocate limited bandwidth to maximally preserve visual information. We propose AutoSelect, which attaches a lightweight Scorer and Denoiser to a frozen VLM and trains with only the standard next token prediction loss, without auxiliary objectives or extra annotations. During training, a variance preserving noise gate modulates each token's information flow according to its predicted importance so that gradients propagate through all tokens; a diagonal attention Denoiser then recovers the perturbed representations. At inference, only the Scorer and a hard top-K selection remain, adding negligible latency. On ten VLM benchmarks, AutoSelect retains 96.5% of full model accuracy while accelerating LLM prefill by 2.85x with only 0.69 ms overhead, and transfers to different VLM backbones without architecture-specific tuning. Code is available at https://github.com/MedHK23/AutoSelect.

#### 1. 核心问题与整体含义（研究动机和背景）
在视觉语言模型（VLM）中，视觉标记（Visual Tokens）的数量随图像分辨率和视频长度增加而激增，由于自注意力机制的平方复杂度，这成为了推理速度和内存占用的主要瓶颈。虽然现有研究表明视觉标记存在大量冗余，但目前的剪枝方法大多依赖于注意力权重或相似度等局部启发式指标。
本文的核心动机是将视觉标记剪枝重新定义为**容量受限的通信问题**：在固定预算 $K$ 的约束下，如何在全球范围内分配带宽以最大化保留下游推理所需的信息。作者认为，模型本身具备识别重要信息的能力，只需通过合适的机制引导其自主选择。

#### 2. 方法论
AutoSelect 框架在冻结的 VLM 架构中插入了两个轻量级模块：**评分器（Scorer）**和**去噪器（Denoiser）**。
*   **核心思想**：将离散的“保留/丢弃”决策转化为连续的“信息容量调制”。
*   **关键技术细节**：
    *   **可学习评分器与 Soft Top-K**：评分器通过 Transformer 块为每个 token 生成分数，并使用可微的 Soft Top-K 算子将分数极化，使其在训练中逼近硬选择（Hard Top-K）。
    *   **方差保持（VP）噪声门控**：在训练阶段，不直接丢弃 token，而是根据评分 $\alpha_i$ 注入噪声：$\tilde{x}_i = \sqrt{\alpha_i} x_i + \sqrt{1 - \alpha_i} \epsilon_i$。高分 token 保留原信号，低分 token 被各向同性高斯噪声淹没。这种方式保证了梯度的全量传播。
    *   **对角注意力去噪器**：为了防止高分 token 的信息泄露给低分 token，去噪器采用对角掩码注意力，仅对单个 token 进行非线性映射，将其从受扰动空间恢复到 LLM 预期的分布。
    *   **训练目标**：仅使用标准的下一个 token 预测损失（Next Token Prediction Loss），无需任何辅助损失函数或额外标注。
*   **推理流程**：推理时移除去噪器和噪声注入，仅保留评分器并执行硬 Top-K 选择，将选中的 $K$ 个 token 及其原始位置索引输入 LLM。

*   **性能保持**：在 LLaVA-1.5-7B 上，即使剪枝掉 88.9% 的视觉 token（仅保留 64 个），仍能保持 96.5% 的原始性能。

*   **非侵入性**：无需微调 LLM 或视觉编码器，保护了预训练模型的语言先验和视觉表征能力。
*   **理论优雅**：将剪枝建模为噪声门控下的容量受限通信，为 token 缩减提供了新的理论视角。

## 摘要
掩码扩散语言模型 (MDLM) 支持并行 Token 解码，为自回归生成的顺序特性提供了一种极具前景的替代方案。然而，它们的迭代去噪过程计算开销依然很大，因为它在每一步都会重复处理整个序列。我们观察到，在这些扩散步骤中，大多数 Token 的表示保持稳定；只有一小部分（我们称之为显著 Token）对下一次更新有实质性贡献。利用这种时间稀疏性，我们提出了 DyLLM，这是一个无需训练的推理框架，通过选择性地仅计算这些显著 Token 来加速解码。DyLLM 通过测量相邻去噪步骤之间注意力上下文的余弦相似度来识别显著性。它仅对显著 Token 重新计算前馈和注意力操作，而对剩余部分重用缓存的激活值。在多种推理和代码生成基准测试中，DyLLM 实现了高达 9.6 倍的吞吐量提升，同时在很大程度上保留了 LLaDA 和 Dream 等最先进模型的基准准确度。

## Abstract
Masked Diffusion Language Models (MDLMs) enable parallel token decoding, providing a promising alternative to the sequential nature of autoregressive generation. However, their iterative denoising process remains computationally expensive because it repeatedly processes the entire sequence at every step. We observe that across these diffusion steps, most token representations remain stable; only a small subset, which we term salient tokens, contributes meaningfully to the next update. Leveraging this temporal sparsity, we present DyLLM, a training-free inference framework that accelerates decoding by selectively computing only these salient tokens. DyLLM identifies saliency by measuring the cosine similarity of attention contexts between adjacent denoising steps. It recomputes feed-forward and attention operations only for salient tokens while reusing cached activations for the remainder. Across diverse reasoning and code-generation benchmarks, DyLLM achieves up to 9.6x higher throughput while largely preserving the baseline accuracy of state-of-the-art models like LLaDA and Dream.

### 1. 核心问题与整体含义（研究动机和背景）
掩码扩散语言模型（MDLM）如 LLaDA 和 Dream，通过并行 Token 解码为自回归模型（ARLM）提供了高效替代方案。然而，MDLM 存在严重的**计算冗余**：在迭代去噪过程中，由于双向注意力机制的存在，模型在每一步都必须重新处理整个序列（类似于“重复的 Prefill 操作”），导致计算成本极高，尤其是前馈网络（FFN）层占据了主要的运行时间。
论文观察到，在相邻的扩散步骤之间，大多数 Token 的表示是极其稳定的，只有极少数关键 Token（即**显著 Token**）发生了实质性的语义转变。DyLLM 的核心动机就是利用这种**时间稀疏性**，通过只计算显著 Token 来加速推理。

### 2. 方法论：核心思想与关键技术
DyLLM 是一个**无需训练（Training-free）**的推理框架，其核心技术包括：

*   **层自适应显著性机制（Layer-Adaptive Saliency）**：
    *   通过计算相邻去噪步骤 $t$ 和 $t-1$ 之间注意力上下文向量的**余弦相似度**来识别显著 Token。
    *   如果相似度高于设定阈值 $\tau$（即 Token 状态稳定），则跳过该 Token 在当前层的 FFN 计算，直接复用缓存的激活值。
    *   实验发现早期层通常比深层更稳定，该机制能自动适应不同层的稀疏度。
*   **显著性感知近似注意力（Saliency-Aware Approximate Attention）**：
    *   将注意力更新分解为“显著路径”和“非显著路径”。
    *   **显著路径**：对显著 Token 进行精确的行更新（Exact Row Update）。
    *   **非显著路径**：利用显著 Token 的增量（Delta）来近似更新稳定 Token 的上下文，避免了 $O(N^2)$ 的全量计算，将复杂度降低到 $O(N \cdot |A|d)$，其中 $|A|$ 是显著 Token 的数量。
*   **响应优先步骤（Response-only Step）**：
    *   利用 RoPE 的局部性偏差，观察到显著 Token 多集中在响应部分。DyLLM 周期性地处理全序列，而在中间步骤仅处理响应 Token，进一步提升吞吐量。

## 摘要
现有提高大型视觉语言模型 (LVLM) 效率的方法主要基于视觉 token 削减的概念。然而，这种方法会产生信息瓶颈，从而损害性能，特别是在需要细粒度理解和推理的挑战性任务上。在这项工作中，我们通过引入 VISion On Request (VISOR) 来挑战这一范式，这是一种在不丢弃视觉信息的情况下降低推理成本的方法。VISOR 并非压缩图像，而是通过使图像和文本 token 之间的交互稀疏化来提高效率。具体而言，语言模型通过一组经过策略性布置的小规模注意力层来关注全套高分辨率视觉 token：通用的视觉上下文由文本-图像之间的高效交叉注意力提供，而少量布置得当且动态选择的自注意力层则负责细化视觉表示本身，从而在需要时实现复杂的、高分辨率的推理。基于这一原则，我们首先通过改变自注意力层的数量，在不同的计算预算范围内训练一个单一的通用网络，然后引入一种轻量级的策略机制，根据每个样本的复杂度动态分配视觉计算资源。大量实验表明，VISOR 在大幅降低计算成本的同时，在各种基准测试中达到或超过了最先进的结果，并且在需要详细视觉理解的挑战性任务中表现出色。

## Abstract
Existing approaches for improving the efficiency of Large Vision-Language Models (LVLMs) are largely based on the concept of visual token reduction. This approach, however, creates an information bottleneck that impairs performance, especially on challenging tasks that require fine-grained understanding and reasoning. In this work, we challenge this paradigm by introducing VISion On Request (VISOR), a method that reduces inference cost without discarding visual information. Instead of compressing the image, VISOR improves efficiency by sparsifying the interaction between image and text tokens. Specifically, the language model attends to the full set of high-resolution visual tokens through a small, strategically placed set of attention layers: general visual context is provided by efficient cross-attention between text-image, while a few well-placed and dynamically selected self-attention layers refine the visual representations themselves, enabling complex, high-resolution reasoning when needed. Based on this principle, we first train a single universal network on a range of computational budgets by varying the number of self-attention layers, and then introduce a lightweight policy mechanism that dynamically allocates visual computation based on per-sample complexity. Extensive experiments show that VISOR drastically reduces computational cost while matching or exceeding state-of-the-art results across a diverse suite of benchmarks, and excels in challenging tasks that require detailed visual understanding.

*   **研究动机**：大型视觉语言模型（LVLM）的推理成本随图像分辨率增加而剧增，主要压力来自大量的视觉 Token。
*   **现有局限**：目前主流的提效方案是“Token 削减/压缩”（Token Reduction），即在输入 LLM 前或在层间丢弃/合并视觉 Token。作者通过实验发现，这种方法会造成不可逆的信息瓶颈，导致模型在需要细粒度理解（如 OCR、文档分析、复杂推理）的“硬任务”上性能大幅下降。
*   **核心目标**：在不丢弃任何视觉信息的前提下，通过稀疏化、动态化的图文交互来降低计算成本，实现“按需分配”视觉计算资源。

### 2. 方法论（VISOR）
*   **核心思想**：将 LLM 内部的计算解耦为“纯文本处理”、“文本-图像交互（交叉注意力）”和“图像表示更新（自注意力）”。
*   **关键技术细节**：
    *   **高效交叉注意力（Cross-Attention, CA）**：在大多数层中，文本 Token 仅通过交叉注意力查询视觉信息，而不更新视觉 Token 本身。这种操作计算量远低于全自注意力。
    *   **选择性自注意力（Self-Attention, SA）**：仅在少数战略性位置执行包含视觉 Token 的全自注意力，用于提取和细化高阶视觉特征。
    *   **通用模型训练（Universal Training）**：训练一个单一模型，使其在训练过程中随机采样不同的 SA 层配置，从而能够适配从“零 SA 层”到“全 SA 层”的多种计算预算。
    *   **动态路由机制（Adaptive Inference）**：引入一个轻量级的 MLP 策略网络（Router），根据输入样本的复杂度（如简单问答 vs. 复杂图表分析），在推理时动态决定开启哪些 SA 层。
    *   **位置编码优化**：采用 1D 深度卷积层实现条件位置编码，以保留高分辨率图像的空域信息。

### 6. 主要结论与发现
*   **性能领先**：VISOR 在 Hard 任务上的表现显著优于所有 Token 削减类方法，甚至在某些配置下超过了全量计算的 Baseline。

*   **任务依赖性**：实验证实，简单任务仅需 0-2 层 SA 即可解决，而复杂任务则需要 7 层以上 SA 来细化特征。

### 7. 优点与亮点
*   **范式创新**：打破了“提效必须减 Token”的固有思维，证明了“稀疏交互”是保留细粒度信息的更优解。
*   **动态自适应**：通过路由机制实现了样本级的计算分配，真正做到了“难题多算，易题少算”。
*   **正交性强**：VISOR 是一种架构级的改进，可以与现有的 Token 剪枝、模型量化等技术完美兼容。
*   **通用性**：一个模型权重即可支持多种计算预算，便于在不同算力的设备上灵活部署。

### 8. 不足与局限
*   **KV Cache 压力**：虽然 FLOPs 大幅下降，但由于保留了全部视觉 Token，推理时的 KV Cache 内存占用并未像 Token 削减方法那样显著减少。
*   **额外参数**：引入交叉注意力层会增加约 7.5% 的参数量（主要是线性投影层）。
*   **路由开销**：虽然路由网络很轻量，但在极小规模模型或极高并发场景下，动态路由本身可能带来微小的调度延迟。
*   **训练复杂性**：相比训练无关的剪枝方法，VISOR 需要两阶段的微调过程。

## 摘要
推理诱导的视觉语言模型（VLMs）通过文本推理推动了图像质量评估（IQA）的发展，但其标量分数往往缺乏敏感性，并会坍缩为少数几个值，即所谓的离散坍缩。我们提出了 ME-IQA，这是一个即插即用的、测试时记忆增强重排序框架。它 (i) 构建了一个记忆库，并利用推理摘要检索语义和感知对齐的邻居；(ii) 将 VLM 重新构建为概率比较器，以获取两两偏好概率，并在 Thurstone's Case V 模型下将该序数证据与初始分数进行融合；以及 (iii) 执行门控反思并巩固记忆，以改进未来的决策。这产生了更密集、对失真敏感的预测，并缓解了离散坍缩。在多个 IQA 基准测试上的实验表明，与强大的推理诱导 VLM 基准、现有的非推理 IQA 方法以及测试时缩放替代方案相比，该方法取得了持续的性能提升。

## Abstract
Reasoning-induced vision-language models (VLMs) advance image quality assessment (IQA) with textual reasoning, yet their scalar scores often lack sensitivity and collapse to a few values, so-called discrete collapse. We introduce ME-IQA, a plug-and-play, test-time memory-enhanced re-ranking framework. It (i) builds a memory bank and retrieves semantically and perceptually aligned neighbors using reasoning summaries, (ii) reframes the VLM as a probabilistic comparator to obtain pairwise preference probabilities and fuse this ordinal evidence with the initial score under Thurstone's Case V model, and (iii) performs gated reflection and consolidates memory to improve future decisions. This yields denser, distortion-sensitive predictions and mitigates discrete collapse. Experiments across multiple IQA benchmarks show consistent gains over strong reasoning-induced VLM baselines, existing non-reasoning IQA methods, and test-time scaling alternatives.

### 1. 核心问题与研究动机
*   **核心问题**：现有的推理诱导型视觉语言模型（VLM）在图像质量评估（IQA）中存在**“离散坍缩”（Discrete Collapse）**现象。即模型倾向于输出少数几个离散的标量分数（如 3.0, 4.0），导致对细微失真不敏感，无法捕捉连续的感知差异。
*   **研究背景**：虽然推理型 VLM（先生成推理再打分）比直接回归模型泛化性更好，但由于 VLM 预训练目标是生成离散 Token 而非连续感知量，导致其在数值预测上表现粗糙。
*   **整体含义**：本文提出 ME-IQA 框架，旨在通过引入“记忆”和“重排序”机制，将 VLM 从一个粗糙的打分器转变为一个精细的感知比较器，从而在测试阶段（Test-time）修复得分分布并提升精度。

### 2. 方法论
ME-IQA 是一个即插即用的测试时增强框架，包含以下核心步骤：
*   **混合记忆库（Hybrid Memory Bank）**：
    *   **锚点记忆（AM）**：离线构建，包含带标签的样本，提供全局稳定的质量参考。
    *   **对比记忆（CM）**：在线动态增长，存储测试过程中遇到的困难或新颖案例，增强局部判别力。
*   **推理感知检索（Reasoning-aware Retrieval）**：
    *   利用 VLM 生成的推理摘要（而非原始图像特征）作为检索键，通过文本嵌入匹配语义和感知上最接近的邻居样本。
*   **基于瑟斯顿模型的重排序（Re-ranking via Thurstone’s Model）**：
    *   **概率比较**：将 VLM 作为比较器，计算待测图像与检索到的 $K$ 个邻居之间的两两偏好概率。
    *   **证据融合**：采用瑟斯顿 Case V 模型，将初始标量分数作为先验，结合两两比较的序数证据，通过最小化二元交叉熵损失（带二次正则项）来优化最终得分。
*   **门控反思与巩固（Gated Reflection & Consolidation）**：
    *   如果重排序后的分数与初始分数差异超过阈值 $\epsilon$，触发 VLM 进行自我反思，修正质量描述并将其存入对比记忆库，供后续查询使用。

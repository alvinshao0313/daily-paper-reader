## 摘要
多模态大语言模型面临严重的计算和内存瓶颈，因为视觉 Token 的数量远超文本 Token。虽然近期的方法采用投影器（projector）模块将视觉 Token 对齐并压缩为文本对齐的特征，但它们通常依赖于固定的启发式方法，限制了在不同场景下的适应性。在本文中，我们首次提出了查询引导混合投影器（QMoP），这是一个新颖且灵活的框架，通过三个协作分支自适应地压缩视觉 Token：(1) 基于池化的分支，用于提取粗粒度的全局语义；(2) 重采样器（resampler）分支，用于提取高层语义表示；(3) 基于剪枝的分支，用于细粒度的 Token 选择以保留关键视觉细节。为了自适应地协调这些分支，我们引入了查询引导路由（QGR），它根据视觉输入和文本查询动态地选择不同分支的输出并分配权重。设计了一种混合专家（MoE）风格的融合机制来聚合输出，在抑制噪声的同时利用每种策略的优势。为了系统地评估视觉 Token 压缩的效果，我们还开发了 VTCBench，这是一个专门用于评估视觉 Token 压缩引起的信息损失的基准测试。大量实验表明，尽管依赖于基础压缩模块，QMoP 的性能仍优于强基准模型，并在内存、计算和推理时间方面实现了显著节省。

## Abstract
Multimodal large language models suffer from severe computational and memory bottlenecks, as the number of visual tokens far exceeds that of textual tokens. While recent methods employ projector modules to align and compress visual tokens into text-aligned features, they typically depend on fixed heuristics that limit adaptability across diverse scenarios. In this paper, we first propose Query Guided Mixture-of-Projector (QMoP), a novel and flexible framework that adaptively compresses visual tokens via three collaborative branches: (1) a pooling-based branch for coarse-grained global semantics, (2) a resampler branch for extracting high-level semantic representations, and (3) a pruning-based branch for fine-grained token selection to preserve critical visual detail. To adaptively coordinate these branches, we introduce the Query Guided Router (QGR), which dynamically selects and weights the outputs from different branches based on both visual input and textual queries. A Mixture-of-Experts-style fusion mechanism is designed to aggregate the outputs, harnessing the strengths of each strategy while suppressing noise. To systematically evaluate the effects of Visual Token Compression, we also develop VTCBench, a dedicated benchmark for evaluating the information loss induced by visual token compression. Extensive experiments demonstrate that despite relying on fundamental compression modules, QMoP outperforms strong baselines and delivers significant savings in memory, computation, and inference time.

#### 1. 核心问题与整体含义
多模态大语言模型（MLLM）在处理图像时，视觉编码器生成的视觉 Token 数量通常远超文本 Token（如 LLaVA-1.5 产生 576 个 Token），这导致了巨大的计算开销和内存瓶颈。现有的视觉 Token 压缩方法（如池化、重采样、剪枝）大多采用固定的启发式规则，无法根据不同的图像内容或用户查询自适应地调整压缩策略。本文旨在提出一种灵活的框架，能够根据视觉输入和文本查询动态选择最优的压缩路径，在大幅减少 Token 数量的同时保持甚至提升模型性能。

#### 2. 方法论
论文提出了 **QMoP (Query Guided Mixture-of-Projector)** 框架，其核心组件包括：
*   **三分支协作架构**：
    1.  **池化分支 (Pooling)**：利用局部注意力机制聚合相邻空间 Token，保留粗粒度的全局场景结构。
    2.  **重采样分支 (Resampler)**：通过可学习的查询（Query）和交叉注意力机制提取高层语义特征。
    3.  **剪枝分支 (Pruning)**：基于重要性（视觉注意力）和相关性（与文本查询的距离）评分，保留关键的细粒度细节 Token。
*   **查询引导路由 (QGR)**：
    *   输入：视觉编码器的 `vCLS` Token 和 CLIP 文本编码器的 `tEOS` Token。
    *   逻辑：将两者拼接后通过多层感知机（MLP）和 Softmax 生成三个分支的权重系数 $\{\alpha_1, \alpha_2, \alpha_3\}$。
*   **混合专家（MoE）风格融合**：在推理时，根据路由权重选择表现最好的两个分支进行加权融合，抑制不相关分支的噪声。
*   **训练策略**：采用两阶段训练。第一阶段联合预训练三个投影器分支；第二阶段激活 QGR 进行联合微调，并引入 Gumbel 噪声和温度退火机制以防止路由塌陷。

#### 6. 主要结论与发现
*   **自适应性至关重要**：不同的压缩策略在不同任务上各有优劣（如池化擅长全局，剪枝擅长细节），QMoP 的动态路由能有效结合各方优势。
*   **高效压缩**：QMoP 将视觉 Token 从 576 压缩至 144 时，性能（100.73%）甚至略优于未压缩的基准模型，同时显著降低了推理延迟和显存占用。
*   **性能拐点**：当 Token 压缩至 36 个以下时，性能会出现明显下降，说明过度压缩会导致关键语义丢失。

#### 7. 优点
*   **创新性架构**：首次将 MoE 思想引入视觉投影器，实现了查询引导的动态压缩。
*   **诊断工具**：提出的 VTCBench 填补了视觉 Token 压缩缺乏专用评估工具的空白。
*   **实用性强**：在不改变 LLM 结构的前提下，作为“即插即用”的模块显著提升了多模态推理效率。

#### 8. 不足与局限
*   **路由开销**：虽然 QGR 相比 LLM 计算量很小，但引入了额外的 CLIP 文本编码器推理步骤来获取文本特征。
*   **训练复杂性**：相比单一投影器，三分支结构和路由器的联合训练需要更精细的超参数调节（如温度退火和噪声处理）。
*   **极端压缩限制**：在极低 Token 数量（如 4 或 16）下，模型依然难以处理复杂的细粒度任务，这反映了信息压缩的固有极限。

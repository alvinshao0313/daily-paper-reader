## 摘要
多模态大语言模型 (MLLMs) 展示了令人印象深刻的跨模态能力，但其庞大的规模给部署带来了巨大挑战。知识蒸馏 (KD) 是压缩这些模型的一种极具前景的方案，但现有方法主要依赖于静态的逐词对齐 (next-token alignment)，忽略了动态的 Token 交互，而这些交互蕴含了多模态理解与生成的关键能力。为此，我们提出了 Align-TI，这是一个从 Token 交互视角设计的创新 KD 框架。我们的方法源于以下洞察：MLLMs 依赖于两种主要的交互：用于提取相关视觉信息的视觉-指令 Token 交互，以及用于连贯生成的响应内 Token 交互。据此，Align-TI 引入了两个组件：IVA 通过对齐显著视觉区域，使学生模型能够模仿教师模型提取指令相关视觉信息的能力；TPA 通过对齐序列化的 Token 到 Token 转移概率，捕捉教师模型的动态生成逻辑。大量实验证明了 Align-TI 的优越性。值得注意的是，我们的方法相比于基础 KD (Vanilla KD) 实现了 2.6% 的相对提升，且蒸馏后的 Align-TI-2B 甚至比规模大得多的 LLaVA-1.5-7B 高出 7.0%，为训练参数高效的 MLLMs 建立了一个新的最先进蒸馏框架。代码可在 https://github.com/lchen1019/Align-TI 获取。

## Abstract
Multimodal Large Language Models (MLLMs) demonstrate impressive cross-modal capabilities, yet their substantial size poses significant deployment challenges. Knowledge distillation (KD) is a promising solution for compressing these models, but existing methods primarily rely on static next-token alignment, neglecting the dynamic token interactions, which embed essential capabilities for multimodal understanding and generation. To this end, we introduce Align-TI, a novel KD framework designed from the perspective of Token Interactions. Our approach is motivated by the insight that MLLMs rely on two primary interactions: vision-instruction token interactions to extract relevant visual information, and intra-response token interactions for coherent generation. Accordingly, Align-TI introduces two components: IVA enables the student model to imitate the teacher's instruction-relevant visual information extract capability by aligning on salient visual regions. TPA captures the teacher's dynamic generative logic by aligning the sequential token-to-token transition probabilities. Extensive experiments demonstrate Align-TI's superiority. Notably, our approach achieves $2.6\%$ relative improvement over Vanilla KD, and our distilled Align-TI-2B even outperforms LLaVA-1.5-7B (a much larger MLLM) by $7.0\%$, establishing a new state-of-the-art distillation framework for training parameter-efficient MLLMs. Code is available at https://github.com/lchen1019/Align-TI.

### 1. 核心问题与整体含义（研究动机和背景）
多模态大语言模型（MLLM）虽然性能强大，但巨大的参数量导致部署成本极高。现有的知识蒸馏（KD）方法主要采用**静态的逐词对齐（Next-Token Alignment）**，即让学生模型在给定的标准前缀下模仿教师模型的输出概率分布。
*   **研究动机**：作者发现这种静态对齐忽略了模型内部的**动态 Token 交互**。这些交互包含两个关键阶段：
    1.  **预填充阶段（Prefilling）**：视觉 Token 与指令 Token 的交互，决定了模型提取关键视觉信息的能力。
    2.  **解码阶段（Decoding）**：响应 Token 内部的交互，决定了生成的连贯性和逻辑推理能力。
*   **核心目标**：通过显式建模这些 Token 交互，提升知识从大模型（教师）向小模型（学生）迁移的效率。

### 2. 方法论：Align-TI 框架
论文提出了 **Align-TI** 框架，包含两个核心组件：

*   **指令感知视觉对齐 (IVA, Instruction-aware Vision Alignment)**：
    *   **核心思想**：并非所有视觉 Token 都同等重要。IVA 让学生模型模仿教师模型在特定指令下的视觉关注点。
    *   **关键技术 (IRS)**：引入**指令相关性得分 (Instruction-Relevance Score, IRS)**。通过计算不同指令下注意力图的余弦相似度，自动识别出语义对齐最清晰的 Transformer 层（通常在模型中后部），并以此层的注意力权重作为指导，让学生模型在关键视觉区域进行对齐。
*   **转移概率对齐 (TPA, Transition Probability Alignment)**：
    *   **核心思想**：缓解“曝光偏差（Exposure Bias）”问题。传统 KD 只对齐标准数据路径上的分布，而 TPA 对齐的是 **Token 到 Token 的转移概率矩阵**。
    *   **算法流程**：在训练时，从学生模型的分布中采样多个候选 Token，并要求学生模型在这些“非标准”路径上的预测概率也与教师模型保持一致。为了提高效率，作者设计了一种**丝带注意力掩码 (Ribbon Attention Mask)**，实现在单次前向传播中并行计算多个路径的转移概率。

### 6. 主要结论与发现
*   **性能突破**：Align-TI-2B 模型在综合评分上达到了 72.2%，不仅远超同尺寸模型，甚至比 **LLaVA-1.5-7B 高出 7.0%**。
*   **交互的重要性**：实验证明，对齐 Token 间的交互动态比单纯对齐输出概率分布更能捕捉教师模型的深层推理机制。
*   **缓解偏差**：TPA 有效降低了曝光偏差（%ExAccErr 显著下降），使学生模型在自主生成时的分布更接近教师模型。

### 7. 优点（亮点）
*   **视角新颖**：从静态对齐转向动态交互对齐，抓住了 MLLM 理解与生成的本质。
*   **原则性层选择**：IRS 指标为“哪一层最适合做蒸馏指导”提供了量化依据，而非经验性选择。
*   **训练效率优化**：通过丝带注意力掩码解决了转移概率对齐带来的计算开销问题，使其在实际训练中可行。

### 8. 不足与局限
*   **模态覆盖**：目前仅验证了图像-文本模态，尚未扩展到视频或音频等多模态场景。
*   **计算开销**：虽然有优化，但 Align-TI 的训练时长仍比基础 KD 高出约 40%，且显存占用略有增加。
*   **架构依赖**：实验主要基于 Qwen 系列架构，虽然在 MobileLLaMA 上做了验证，但对于非 Transformer 架构（如 Mamba）的适用性尚未探讨。

## 摘要
多模态大语言模型 (MLLMs) 在通过所有 LLM 层处理大量视觉 token 时会产生巨大的计算开销。先前的剪枝方法要么在 LLM 之前操作，由于编码器-投影器设计的多样性而限制了通用性；要么在 LLM 内部使用与 FlashAttention 不兼容的启发式方法。我们采取了一种不同的方法：与其识别不重要的 token，不如将 LLM 本身视为压缩的最佳指南。观察到深层自然地传递视觉到文本的信息，我们引入了注意力驱动的自压缩 (ADSC)，这是一种简单且广泛适用的方法，仅利用 LLM 的注意力机制逐步减少视觉 token。我们的方法在选定的层应用均匀的 token 下采样，形成瓶颈，促使模型将信息重新组织并压缩到剩余的 token 中。它不需要分数计算、辅助模块或注意力修改，并且与 FlashAttention 完全兼容。应用于 LLaVA-1.5 时，ADSC 将 FLOPs 降低了 53.7%，峰值 KV 缓存内存降低了 56.7%，同时保留了原始模型 98.2% 的性能。在多个基准测试中，它在效率和准确性方面均优于先前的剪枝方法。至关重要的是，在高压缩率下，我们的方法保持稳健，而基于启发式的技术则会急剧下降。

## Abstract
Multimodal Large Language Models (MLLMs) incur significant computational cost from processing numerous vision tokens through all LLM layers. Prior pruning methods operate either before the LLM, limiting generality due to diverse encoder-projector designs or within the LLM using heuristics that are incompatible with FlashAttention. We take a different approach: rather than identifying unimportant tokens, we treat the LLM itself as the optimal guide for compression. Observing that deeper layers naturally transmit vision-to-text information, we introduce Attention-Driven Self-Compression (ADSC), a simple, broadly applicable method that progressively reduces vision tokens using only the LLM's attention mechanism. Our method applies uniform token downsampling at selected layers, forming bottlenecks that encourage the model to reorganize and compress information into the remaining tokens. It requires no score computation, auxiliary modules, or attention modification, and remains fully compatible with FlashAttention. Applied to LLaVA-1.5, ADSC reduces FLOPs by 53.7% and peak KV-cache memory by 56.7%, while preserving 98.2% of the original model performance. Across multiple benchmarks, it outperforms prior pruning approaches in both efficiency and accuracy. Crucially, under high compression ratios, our method remains robust while heuristic-based techniques degrade sharply.

### 1. 核心问题与整体含义（研究动机和背景）
多模态大语言模型（MLLM）在处理高分辨率或长视频输入时，视觉 Token 的数量往往远超文本 Token，导致了巨大的计算开销和内存占用（KV Cache）。
*   **现有方法的局限性：** 
    *   **LLM 前置压缩：** 依赖特定的编码器-投影器设计，通用性差，且可能在进入 LLM 前丢失关键语义。
    *   **LLM 内部剪枝：** 大多依赖注意力分数（Attention Scores）来识别重要 Token。这种方式需要显式生成完整的注意力矩阵，与 **FlashAttention** 等高效硬件加速算子不兼容，导致实际推理速度提升有限。
*   **研究目标：** 提出一种不依赖注意力分数、完全兼容 FlashAttention、且能让模型“自主学习”如何压缩信息的通用视觉 Token 削减方案。

### 2. 方法论：注意力驱动的自压缩 (ADSC)
*   **核心思想：** 将 LLM 视为一个天然的信息压缩器。通过在特定层强制减少视觉 Token 数量（形成信息瓶颈），利用 Transformer 的注意力机制促使模型将即将删除的 Token 信息“迁移”到保留的 Token 或文本 Token 中。
*   **关键技术细节：**
    *   **均匀下采样（Uniform Downsampling）：** 在预定义的剪枝层（如第 8、16、24 层），按照固定步长均匀保留视觉 Token，丢弃其余部分。这种方式不涉及复杂的评分计算。
    *   **信息迁移机制：** 由于采用了因果注意力（Causal Attention），文本 Token 位于序列末尾，能够吸收所有视觉信息。模型在训练过程中会学习利用位置编码，将重要视觉特征整合进幸存的 Token 中。
    *   **反向课程学习（Reverse Curriculum）：** 在训练阶段，先采用比目标压缩率更严苛的比例进行训练（如目标的 80% 预算），随后逐渐放宽至目标比例。这能迫使模型在极端瓶颈下建立更强的压缩能力。
    *   **参数高效微调：** 仅更新 LoRA 适配器和投影层（Projector），冻结视觉编码器和 LLM 主干，以保持原始模型的理解能力。

*   **数据集：** 涵盖 6 个主流视觉语言基准：

*   **实验规模：** 论文在 6 个不同维度的基准测试上进行了详尽对比，并针对三种不同的压缩比例（66.7% 到 88.9% 的削减率）进行了测试。
*   **消融实验：** 
    *   对比了全参数微调与 LoRA 微调的效果。

*   **客观性：** 实验涵盖了从感知到推理、从幻觉到 OCR 的多种任务，对比基线均为近年来的代表性工作，实验设计较为客观、公平。

### 6. 主要结论与发现
*   **高效性：** 在保留 192 个 Token 时，ADSC 减少了 **53.7% 的 FLOPs** 和 **56.7% 的 KV 缓存**，同时保留了 **98.2%** 的原始性能。
*   **高压缩下的鲁棒性：** 在极端情况下（仅保留 64 个 Token，削减率约 90%），传统启发式方法（如 ToMe, FastV）性能大幅崩溃（下降 30+ 点），而 ADSC 仍能保留 **95.1%** 的性能。
*   **自适应压缩：** 实验证明，即使采用简单的均匀下采样，模型也能通过注意力机制自主将关键信息路由到幸存 Token 中，无需显式的重要性评分。

### 7. 优点（亮点）
*   **极简且通用：** 无需修改模型架构，无需引入额外的预测模块或复杂的聚类算法。
*   **硬件友好：** 完全兼容 FlashAttention，这是目前大多数剪枝方法难以做到的，具有极高的实际部署价值。
*   **训练策略创新：** 提出的反向课程学习显著提升了模型在极低 Token 预算下的表现。

### 8. 不足与局限
*   **依赖微调：** 与 FastV 等“即插即用”的推理加速方法不同，ADSC 需要经过监督微调（SFT）才能让模型适应 Token 减少后的分布。
*   **空间先验限制：** 均匀下采样虽然简单，但可能在某些极端长宽比或局部细节极度密集的图像上不如动态采样精确。
*   **任务覆盖：** 实验主要集中在静态图像任务，尚未在视频理解（Video-LLM）等更长序列的任务中进行大规模验证。

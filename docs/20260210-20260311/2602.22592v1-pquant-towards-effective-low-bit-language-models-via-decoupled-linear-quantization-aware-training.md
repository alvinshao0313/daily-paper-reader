## 摘要
从零开始的量化感知训练（Quantization-Aware Training from scratch）已成为构建具有极低比特权重（低于 2 比特）的高效大语言模型（LLMs）的一种极具前景的方法，这能为边缘部署提供显著优势。然而，现有方法仍无法达到令人满意的准确率和可扩展性。在这项工作中，我们发现“参数民主化效应”（parameter democratization effect）是一个关键瓶颈：所有参数的敏感度趋于同质化，严重限制了模型的表达能力。为了解决这一问题，我们提出了 pQuant，该方法通过将线性层拆分为两个专门的分支来解耦参数：一个用于高效计算的主导 1 比特分支，以及一个专门用于保留最敏感参数的紧凑高精度分支。通过定制的特征缩放，我们显式地引导模型将敏感参数分配到高精度分支。此外，我们将该分支扩展为多个稀疏激活的专家（experts），从而实现高效的容量扩展。广泛的实验表明，我们的 pQuant 在极低比特量化方面达到了当前最先进的性能。

## Abstract
Quantization-Aware Training from scratch has emerged as a promising approach for building efficient large language models (LLMs) with extremely low-bit weights (sub 2-bit), which can offer substantial advantages for edge deployment. However, existing methods still fail to achieve satisfactory accuracy and scalability. In this work, we identify a parameter democratization effect as a key bottleneck: the sensitivity of all parameters becomes homogenized, severely limiting expressivity. To address this, we propose pQuant, a method that decouples parameters by splitting linear layers into two specialized branches: a dominant 1-bit branch for efficient computation and a compact high-precision branch dedicated to preserving the most sensitive parameters. Through tailored feature scaling, we explicitly guide the model to allocate sensitive parameters to the high-precision branch. Furthermore, we extend this branch into multiple, sparsely-activated experts, enabling efficient capacity scaling. Extensive experiments indicate our pQuant achieves state-of-the-art performance in extremely low-bit quantization.

### 1. 核心问题与研究动机
*   **研究背景**：为了在边缘设备上部署大语言模型（LLM），极低比特（低于 2-bit）量化成为研究热点。目前主要有两条路径：后训练量化（PTQ）和从零开始的量化感知训练（QAT-Scratch）。
*   **核心问题**：现有的 1-bit QAT-Scratch 模型（如 BitNet）存在两个瓶颈：一是准确率与全精度（FP16）模型相比仍有较大差距；二是扩展性差，模型规模增加带来的性能提升呈次线性，远低于 FP16 模型。
*   **关键发现**：作者提出了**“参数民主化效应”**（Parameter Democratization）的概念。在全精度模型中，参数敏感度分布极不均匀（少数参数极度敏感）；但在 1-bit 模型中，所有参数的敏感度趋于同质化，这种“平等”限制了模型对关键特征的表达能力。

### 2. 方法论：pQuant 核心思想与技术细节
pQuant 的核心是通过结构化解耦来打破参数敏感度的同质化：
*   **解耦线性层（Decoupled Linear Layer）**：将传统的线性层拆分为两个并行分支：
    *   **1-bit 主分支**：承载绝大部分参数，保证计算效率和低内存占用。
    *   **8-bit 紧凑分支**：仅包含极少数参数（约 4%-5%），专门用于捕捉和保留那些对输出影响巨大的“敏感参数”。
*   **特征缩放引导（Feature Scaling）**：引入可学习的标量 $\alpha$（高精度分支）和 $\beta$（1-bit 分支）。在训练初始化时设置 $\alpha \gg \beta$，显式引导梯度优先流向高精度分支，从而迫使模型将最重要的表示分配给 8-bit 路径。
*   **稀疏激活专家扩展（MoE Scaling）**：为了进一步提升容量，将高精度分支扩展为 $N$ 个专家。利用轻量级路由器进行 Top-1 激活，使得模型在推理时仅增加极少的计算开销，却能显著提升模型表达力。
*   **量化方案**：权重采用 Sign 函数量化为 1-bit，激活值采用 AbsMax 量化为 INT8。

*   **训练成本**：作为 QAT-Scratch 方法，其训练成本远高于 PTQ 或微调方法，且由于需要维护 FP16 影子权重，训练时的显存占用较高。

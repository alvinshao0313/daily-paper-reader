## 摘要
全扫描切片图像 (WSIs) 具有分层结构，其诊断信息源于细胞形态、区域组织结构和全局上下文。现有的计算病理学 (CPath) 多模态大语言模型 (MLLMs) 通常将整个 WSI 压缩为单一嵌入，这阻碍了细粒度的证据定位，并忽略了病理学家跨不同尺度综合证据的方式。我们提出了 MLLM-HWSI，这是一种分层 WSI 级 MLLM，它在四个不同尺度上将视觉特征与病理语言对齐：细胞视为单词、补丁视为短语、区域视为句子、WSI 视为段落，以支持可解释的、基于证据的推理。MLLM-HWSI 通过特定尺度的投影器将每个 WSI 分解为多尺度嵌入，并共同执行 (i) 分层对比目标和 (ii) 跨尺度一致性损失，从而保持从细胞到 WSI 的语义连贯性。我们计算具有诊断相关性的补丁，并利用轻量级的细胞-细胞注意力融合 (CCAF) Transformer 将分割后的细胞嵌入聚合为每个补丁的紧凑细胞标记。投影后的多尺度标记与文本标记融合，并输入到经过指令微调的 LLM 中，用于开放式推理、视觉问答 (VQA)、报告和字幕生成任务。经过三个阶段的训练，MLLM-HWSI 在六个 CPath 任务的 13 个 WSI 级基准测试中取得了新的 SOTA 结果。通过将语言与多尺度视觉证据对齐，MLLM-HWSI 提供了准确、可解释的输出，模拟了诊断工作流程并推进了全面的 WSI 理解。代码可在 GitHub 获取：https://github.com/BasitAlawode/HWSI-MLLM。

## Abstract
Whole Slide Images (WSIs) exhibit hierarchical structure, where diagnostic information emerges from cellular morphology, regional tissue organization, and global context. Existing Computational Pathology (CPath) Multimodal Large Language Models (MLLMs) typically compress an entire WSI into a single embedding, which hinders fine-grained grounding and ignores how pathologists synthesize evidence across different scales. We introduce \textbf{MLLM-HWSI}, a Hierarchical WSI-level MLLM that aligns visual features with pathology language at four distinct scales, cell as word, patch as phrase, region as sentence, and WSI as paragraph to support interpretable evidence-grounded reasoning. MLLM-HWSI decomposes each WSI into multi-scale embeddings with scale-specific projectors and jointly enforces (i) a hierarchical contrastive objective and (ii) a cross-scale consistency loss, preserving semantic coherence from cells to the WSI. We compute diagnostically relevant patches and aggregate segmented cell embeddings into a compact cellular token per-patch using a lightweight \textit{Cell-Cell Attention Fusion (CCAF)} transformer. The projected multi-scale tokens are fused with text tokens and fed to an instruction-tuned LLM for open-ended reasoning, VQA, report, and caption generation tasks. Trained in three stages, MLLM-HWSI achieves new SOTA results on 13 WSI-level benchmarks across six CPath tasks. By aligning language with multi-scale visual evidence, MLLM-HWSI provides accurate, interpretable outputs that mirror diagnostic workflows and advance holistic WSI understanding. Code is available at: \href{https://github.com/BasitAlawode/HWSI-MLLM}{GitHub}.

### 2. 方法论：核心思想与关键技术
*   **核心思想**：将 WSI 的层级结构类比为语言结构——**细胞即单词、补丁即短语、区域即句子、WSI 即段落**。
*   **关键技术细节**：
    1.  **层级多尺度编码器**：采用 CellViT（细胞级）、CONCH（补丁级）、HIPT（区域级）和专门预训练的 ViT（WSI 级）提取特征。
    2.  **语义补丁过滤（SPF）**：利用视觉相似度和报告关键词相关性，从数百万个补丁中筛选出最具诊断价值的异质性补丁，降低计算冗余。
    3.  **细胞-细胞注意力融合（CCAF）**：使用轻量级 Transformer 聚合补丁内的细胞嵌入，生成紧凑的细胞特征标记。
    4.  **损失函数设计**：
        *   **层级对比对齐损失 ($L_{HCA}$)**：在每个尺度上分别进行图文对齐。
        *   **跨尺度一致性损失 ($L_c$)**：强制相邻层级（如补丁与区域）之间的语义保持连贯，防止语义漂移。
*   **训练流程**：分为三个阶段：(1) 层级跨模态对齐预训练；(2) 特征空间投影层对齐；(3) 任务特定指令微调（使用 LoRA 技术）。

*   **数据集与场景**：涵盖了 13 个公开数据集（如 PANDA, EBRAINS, TCGA-OT, BRACS, UBC-Ocean, IMP-CRC 等），涉及前列腺、脑、乳腺、卵巢、结直肠等多种癌症。
*   **评估任务（Benchmark）**：

    3.  **报告/字幕生成**：评估语言描述的准确性与临床一致性。

*   **对比方法**：与 24 种 SOTA 模型对比，包括通用多模态模型（GPT-4V, LLaVA）和病理专用模型（SlideChat, WSI-LLaVA, TITAN, CONCH, UNI 等）。

*   **硬件环境**：使用了 **4 台 NVIDIA A100 80GB GPU**。

### 6. 主要结论与发现
*   **性能领先**：MLLM-HWSI 在所有 13 个基准测试中均刷新了 SOTA 记录。例如，在零样本分类中平均准确率比 TITAN 高出 7.3%。

*   **临床一致性**：定性分析显示，模型生成的病理报告在描述核异型性、组织结构和诊断结论方面与人类专家高度接近。

### 7. 优点（亮点）
*   **符合临床逻辑**：首次系统性地将 WSI 的生物学层级与语言学层级一一对应，模拟了病理医生的诊断思维。
*   **创新的过滤机制**：SPF 模块有效解决了 gigapixel 图像输入 LLM 时的 Token 爆炸问题，同时保留了关键诊断证据。
*   **强解释性**：通过多尺度对齐，模型能够将文本描述（如“多形性核”）直接关联到具体的视觉层级（细胞级特征）。

### 8. 不足与局限
*   **计算开销**：由于需要运行多个层级的编码器（尤其是细胞分割和多尺度特征提取），其推理时间比单一尺度的模型稍长。
*   **数据依赖**：模型性能高度依赖于高质量的病理报告对齐数据，而这类数据的获取成本极高。
*   **应用限制**：目前主要针对 H&E 染色切片，对于免疫组化（IHC）或特殊染色的泛化能力尚未在文中深入探讨。
*   **实时性**：4.9 分钟/张的推理速度对于极高通量的临床实时诊断仍有优化空间。
